Here is a list of modes that I have been putting together over a period
of time. The numbers are the consecutive intervals in number of scale steps.
It is clear that such a list of bare numbers is in many cases an
oversimplification of things. For instance for an Indian raga it is only
one technical aspect of a whole musical system which can also have
different ascending and descending scale patterns, different stress on
notes, enharmonic variations, etc. So no attempt was made to make the
list complete for Indian ragas. Most of these listed have the same
ascending and descending scale, however if for a particular case none
having that exists, I have incorporated one which omits a note in
either or both cases. All 72 ragas of the South-Indian (Karnatic)
melakarta raga system are in the list indicated with "mela".
North-Indian (Hindustani) equivalent names are given also.
There are also more Japanese pentatonic modes, different forms of basic
ones.
For more information about the concept of mode, there is a lot to find
under "mode" in the New Grove encyclopedia.
The number of modes in the list is 208. Lines beginning with a ! are
comments. Additions are welcome.
