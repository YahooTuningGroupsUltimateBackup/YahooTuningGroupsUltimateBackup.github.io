6/7/2013 8:06:16 AM

The latest Scala version 2.34n can create Lesfip scales.
The definition is here: http://xenharmonic.wikispaces.com/Lesfip+scales
See the new dialog Approximate:Least-squares approximation, or the command APPROXIMATE/LEASTSQUARE. The scales cons*.scl in the scale archive should be useful as target consonances, as well as n-limit diamonds. Remember to delete the 2/1 from them first.

Tempering with the optimal standard val is also a relatively new addition. It's not the same as Gene's definition, because he uses TE error and I have used TOP-RMS error, which is much simpler to calculate. The result is often the same, but it looks like more often higher or lower ETs are the result. See Modify:Temper or
PROJECT/TEMPER/OPT_ET_RMS.

Manuel

Yes but the name of the item has changed. It's now Approximate:Interval approximation because minimax optimization has been added to the dialog. Clicking on Apply or OK does the lesfip or mimafip optimization, but it can also be done one step at a time with the Apply Single Iteration button.
