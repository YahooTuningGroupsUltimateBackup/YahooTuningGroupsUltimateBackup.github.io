If we have a planar temperament in the strict sense, meaning octaves
are the period, and a set of intervals in it, a tictactoe diagram is
one with an X where there is a note and an O where there is not, in a
grid arrangement defined by the generators. If the generators are the
ones given by Hermite reduction, and if the set of notes is the otonal
tetrad, you get a tictactoe diagram showing how the basic 7-limit
chords work, the utonal tetrad being the radial inverse. If we are
representing a linear temperament in a plane, more than one comma
gives the same generators, and so more than one configuration is
useful to know. For instance meantone in the 5-limit plane has both
the 225/224 and the 126/125 configuration, as well as others. Below
are tictactoe diagrams for some 7-limit commas leading to planar
temperaments. This sort of thing is particularly useful for Tonescape,
but I think is generally interesting. It should be clear looking at
this why 225/224 and 2401/2400 are so important as 7-limit planar
temperaments.
