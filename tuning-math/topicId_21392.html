I've added two more categories of rank-2 temperaments to Scala's list in the file lintemp.par.

One is the Wilson type of equal beating temperaments where 5/4 and 3/2 have an equal (or integer proportional) beating rate, called Meta-<name> so to list them in Scala type:
LINEAR/ALL Meta-*
I also included a few 3/1 period based ones with 5/3 and 7/3 equal beating.

And the other one the "golden" versions which have the property that the ratio between large and small step size is equal to the golden ratio phi. To list them in Scala type:
LINEAR/ALL Golden

For some more information, enter the command TIP 146.

There's also a new datafile called series.par, which is used in the dialog
File:New:Recurrent series. You can type a name in the coefficients combobox and this will allow you to select a predefined series. Or take it from the dropdown list. Most of them came from Jacques Dudon, who no doubt has invented hundreds more of them but these are all I could find.
Here you'll find the same names as the Meta temperaments, which means the root
of this polynomial is the generator of that temperament. You can check this, for example select: 1 1 1/2 (Zinith, Meta-Pelog), select Recurrent series under Kind and click Find Polynomial Root. The root is then shown in the main window.
In order to create a recurrent series from this you additionally have to enter initial values under Parameters. More info in the help for POLYNOMIAL/SERIES.
Below is the current contents of series.par.
