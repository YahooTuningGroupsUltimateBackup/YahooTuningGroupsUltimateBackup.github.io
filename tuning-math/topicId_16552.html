I've updated my online temperament finder code at

http://x31eq.com/temper/

to use the updated library for weighted prime limit searches. The main difference you might notice is that it's faster.

I also raised the maximum prime limit. 41-limit searches seem to be in fashion now, although my previous 31-limit cap (following Ptolemy) used to look ridiculously high. So you can now do 41-limit searches. You can even go higher than that and you may at some point discover the new limit to the limit in which case I might raise it again.

And I sneaked in a change to the source code zip file for the new limit, and to fix a bug where the 1.5.2 code didn't work with 1.5.2.
