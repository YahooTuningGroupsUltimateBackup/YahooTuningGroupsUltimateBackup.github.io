I'd like to broaden the meaning of TOP. Instead of top being a Tenney-weighted minimax error tuning, I'd like to say "TOP RMS" for the optimal Tenney-weighted prime-interval RMS. This can work if we define TOP to mean "Tenney-weighted Optimal Prime".

So old TOP error could be new "TOP max".

As well as TOP RMS we can have TOP mean, or TOP MAD if you prefer that.

It doesn't really change the original meaning. There was nothing in "Tenney OPtimal" or "Tempered Octaves Please" to denote a max-abs error anyway. And we can keep plain "TOP" for the max error case. It means the "prime" is redundant for old top, but the simplest calculations use primes anyway.

If we want to talk about Tenney weighting, optimization and primes there aren't many other acronyms anyway. And it's getting to be a pain to have to spell out the TOP RMS each time. It's unfortunate that the word "top" is used to mean a *minimum* but so be it.

Then we could also have "TOPPO" for "TOP with Pure Octaves" which I like. This would refer to any of the above, but with the scale stretched so that octaves are pure.

It's something that'd be nice to have if we can get a consensus on a glossary. Obviously we'll need to get Paul E's blessing so if anybody sees him you can pass this along.
