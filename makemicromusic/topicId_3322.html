Howdy Microtunies:

Remember the announcement of Angelina a few days ago? If not, I'll
refresh. Angelina is a soft synth VST plugin or standalone, which (quoting
the blurb) "uses formant synthesis to generate rich, breathy pads sound".

Angelina is distributed as shareware at a mere $40, and is currently in
Beta. See the page here:

http://bigtick.pastnotecut.org/index.php

and click on Angelina in the left column.

Here is a bit of news for the practicing microtonalists... I've been in
contact with "Big Tick" the author of Angelina and Rainbow soft synths.
Big Tick is currently adding microtonal support to Angelina, using the
same file format as the VAZ line (i.e., the ".tun" file). Yesterday
afternoon I received a pre-alpha copy of Angelina with preliminary
microtonal support. I was able to load ".tun" files that I have used with
VAZ and produce a bit of doodling improvisation with Angelina, which I
uploaded as an example:

http://rm-and-jo.laughingsquid.org/Music/Angelina-15-Tone-Test.mp3

(It's just goofing around, so don't expect music; but it illustrates
Angelina with microtonal support.)

Rumor has it that micrtonal support might eventually make it into Rainbow
as well... Hmmm! Well, I've already signed up to support the effort!

Rick
