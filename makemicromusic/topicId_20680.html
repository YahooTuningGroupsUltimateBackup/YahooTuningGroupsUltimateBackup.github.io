CSE = Custom Scale Editor, a free cross-platform microtonal scale creation utility.

Some notable new features in version 2.3.0 include a 'rationalize' continued fraction engine, and support for exporting and importing tuning files for Native Instruments software Absynth, Kontakt, and Pro53, as well as Max Magic Microtuner, Scala, VAZ/Anamark, Cubase / Nuendo, and Scordatura.
