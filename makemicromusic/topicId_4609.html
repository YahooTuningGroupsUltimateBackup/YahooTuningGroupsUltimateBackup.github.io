On the subject of microtuning in Reaktor, I came across this post on Native Instruments' forums.

21-05-2002
mmclain : Is it possible to microtune Reaktor? For example can reactor be programmed to play a 7 note Just scale? I downloaded the demo and I can't find anything that relates to microtuning. If microtuning is possible can anyone tell me how?

greg : Hi, You can do anything with Reaktor. Conserning microtuning you can make a table so that any incoming note maps into a specific user defined pitch. This can be done in several ways. Thers probably an ensemble to be found in the user forum. sincerely Greg

I know Loffink here uses Reaktor so should be able to confirm this. It's also similar (from what I understand) to the way one would have to go about doing this in Max/MSP - design your own microtuning implementation.

For those who need to work with samples, Reaktor imports Akai format samples (I don't know about others) and supports AIFF, WAV, and SDII format samples in up to 32-bits.

Reaktor 4 has just become available and supports AudioUnits (a step up from VST) and CoreAudio in OS X.

There are three ways to microtune Reaktor.

For equal tempered scales, simply insert a scaling macro between the
pitch control signal and the pitch inputs. The macro Microtune provides
scaling factors for 7 common microtonal scales selectable by virtual
buttons.

For twelve note per octave scales in any tuning, use the Microtune 12
macro. This lets you set the tuning for each note using virtual knobs.

For non-equal scales with more than twelve notes use an event table
module to map input to output values.

Using these modules and macros is not difficult, but adapting existing
Reaktor instruments for microtuning is non-trivial.

For somewhat easier microtuning support in a virtual modular synthesizer
I would recommend VAZ Modular.
