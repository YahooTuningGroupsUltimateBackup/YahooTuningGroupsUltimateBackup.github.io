I have been following the thread on Kontakt, and am rather puzzled
why some people seem to be having problems with this.

A couple of years ago, I figured out how to do this using Logic's EXS.

You can select and edit any sample to be played by any midi note from
any midi note reference; and then adjust the fine tuning for each
midi note (+/- 50 cents) as you like.

Logic Pro 7 has refined the micro(nano?)tuning capability, and can
use scala or many other formats to set up tables e.g. via Max
MicroTuner etc.

Later Logic Pro 7 versions are only available for Mac OSX nowadays,
yet there still seem to be lotsa kraks out there for the old Windoze
versions of Logic 5,
which I hear work well when downloaded from LimeWire or similar.

I hope Apple will eventually just make them legally available, so
that M$ users can get a taste of the benefits of using Logic.

These pages may help if you are exploring other options and the
details of microtuning samples, without spending an arm and a leg.

http://www.lucytune.com/midi_and_keyboard/pitch_bend.html

http://www.apple.com/logicpro/

I have also been experimenting with Vocal Writer shareware (30 day
trial), and found that you can actually ptichbend the vocal tuning
within the application,
as well as by using Logic Pro microtuning + pitch corrections.

http://www.kaelabs.com
