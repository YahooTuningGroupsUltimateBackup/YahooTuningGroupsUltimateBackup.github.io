I've made up some Thummer keyboard maps for use in Scala (Tested with
Scala22): add "Clav_Key_File keymapname.par" to the scala.cfg file,
and place the keymap file in the Scala directory

They're all Equal-tempered meantones, So far I've made 12, 19, 31, and
55, and they are all centered on the (letter) G = Scale degree 0

I typed these in by hand based on whatever P4 P5 and M2 values I could
find online, so there may be mistakes.

http://rozencrantzthesane.tripod.com/scalamaps/index.html

I suppose these are Public Domain. I certainly don't care what you do
with them, but tell me if you use them for anything so I can get a
warm squishy feeling inside.
