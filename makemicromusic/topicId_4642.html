Scala and sysex Commands

Dear group,

By calling Yamaha I found out that the PSR 330 and my PSR 340 do have
the same tone generation card and that the appropriate sysex commands
look like (from a PSR 330 manual):

F043104C0800417FF7

To retune to a new scale would require 12 such commands per channel
at the most. Note: the 00417F part is all that changes. Numbers are
all in hexadecimal. 00 to 0F denotes channel 1 to 16; 41 to 4C points
to all Cs all C#s and so on, ending with 4C for all the Bs; next is
the pitch adjustment: 00 = -63 cents 40 = 0 cents 7F=+63 cents. If
one could set up a table like a spread sheet to help remember the
relationships that would be great. Scala does something like a table
but I can't figure out how. I've tried: SET SYNTH 107 and SET
SYNTH 1
SEND F043104C0800417FF7 without much success. Would someone please
help?

Best wishes,
