Hi,

I've been looking at information on 88-key weighted
controllers / digital pianos on the web and the most tempting of
the lot that I've come across is the Kawai MP9500 with it's
graded, wooden key action.

The most sensibly priced and respectable controller is the new
Fatar SL 880 PRO (€ 699.00), also with graded weighted action.

http://www.arbiterfrance.com/article.php?numref=61
http://www.arbitermt.co.uk/fatar/88notekeyboards.htm
http://fr.audiofanzine.com/news/index/14985

Getting back to the Kawai, the german manual is the only one I
found for the MP9500 and it shows that the instrument has the
foll. temperament options :

Equal / Stretch / Pure M / Pure m / Pytha / Mean / Werk / Kirn / User

The user tempermanent allows you to vary the 12-tone C-B range
within a max of +/- 50 cents with a precision of 1 cent.

Caveat ? Well, the manual seems to indicate that the user
temperament function cannot be accessed via MIDI SysEX. Only
the instrument panel.

Tonart (Key of Temperament)
[Werte: C ~ B]
Eine unbegrenzte Modulation zwischen allen Tonarten wurde erst nach der
Einführung der gleichschwebenden temperierten Stimmung möglich.
Für alle
anderen Stimmungen müssen Sie die Tonart, in der Sie das
entsprechende
Stück spielen wollen, sorgfälltig auswählen.
Zum Beispiel: Falls das Stück in D Dur geschrieben ist, müssen
Sie “D” als
Tonart angeben.
Wenn Temperament auf Equal oder Stretch gestellt ist, wird diese
Seite nicht
angezeigt.

USER Tuning C ~ B
[Werte: -50 ~ +50]
Wenn die Temperatur auf “User” gestellt ist, stellen Sie hier
die Werte für jeden
Halbton ein und erzeugen so Ihre eigene Stimmung.
Diese Seiten werden nur angezeigt, wenn “User” als Temperatur
eingestellt ist.

System Stimmung / SysTune
[Werte: 427.0 ~ 453.0 (Hz)]
Dieser Parameter stellt die Stimmung des MP9500 für den
Kammerton A ein.
