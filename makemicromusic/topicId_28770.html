Can't download it because it ain't finished yet. To run it with Logic, you have to also run reaper and use rewire to send midi back and forth. This is not nearly as big a hassle as it sounds. Reaper is about a 5Mb download, installs quickly, not copy-protected, demo version is free, full version is cheap, demo version actually IS the full version, etc. Reaper is what software is supposed to be. It costs less than a vst wrapper.
