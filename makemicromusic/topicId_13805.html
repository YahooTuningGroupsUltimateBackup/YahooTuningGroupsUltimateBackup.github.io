Hello people.

I just uploaded the latest version of microabc (2006-06-28), a free (GNU GPLed) software to help making microtonal music with ABC tools (abcm2ps, abcpp and abcMIDI).

The new version includes a basic built-in preprocessor (just a macros replacer) which can be used instead of abcpp in some cases, as well as a Sagittal-2.0d8 PostScript font (for microtone accidentals). This is not yet the final version of Sagittal-2 font, it is released for testing purposes and can be subjected to changes (thanks to Dave Keenan and George Secor).

A relatively complete support for Sagittal Notation System is being worked and it will be available soon (maybe in next release). Just now, you can use abcpp and the files `sagittal.fmt', `sagittal-pfb.fmt', `sagittal-mixed.fmt', `sagittal-midi.abh' and `sagittal-midi-12et.abh' to write music in Sagittal notation. See inside `examples/sagittal/'.

Any comments are very welcome.

Best wishes.
