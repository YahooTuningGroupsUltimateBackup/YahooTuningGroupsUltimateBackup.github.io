Hi everyone,
As a devout lurker here, some posts in January ago got me interested
in Kontakt 2 as a microtonal sampler. I began experimenting and was
delighted to find that Kontakt 2 can be scripted to support multiple
instances of fully polyphonic microtonal tunings, each addressable for
realtime retuning via midi cc events. Coding the script for sets of
tunings is tedious, so I wrote a Windows utility to load a group of
Scala tunings and output a tuning script.

It occurred to me that this might be of interest to other Windowholics,
so I got a goofy domain name and a web host to offer it as shareware.
The site is http://www.12equalboresme.com If you have a fast PC and
a budget for Kontakt 2, I hope you give it a try.
Regards,
Robert Strauss
