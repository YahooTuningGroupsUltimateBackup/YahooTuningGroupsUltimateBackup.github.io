Well i haven't been using granulation a lot because it gets too regular in
the long run. Just recently i have used the Audiomulch granulator for pitch
shifting effects-as i can be set to any cent value, i can use several in
cross-mixing which creates a varying complex out of a static note input.

will post some pieces made with this on RS soon.
