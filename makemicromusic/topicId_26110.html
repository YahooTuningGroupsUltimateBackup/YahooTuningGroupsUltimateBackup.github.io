Hey all,

I found this freeware VSTi & discovered it had some microtonal
capabilities. This is not a microtonal instrument per se, but by
tweaking a control in the oscillator section a number of microtonal
scales become available.

http://hercsmusicsystems.com/old_index.html <http://>

Once the plug-in is installed into your host & is up on the screen, you
will see -- in the oscillator section -- a knob called "FOLLOW". The
default setting for 12et is +8. By experiment, I found these settings &
(provisional) results:

+2 = 28et
+3 = 32et
+4 = 24et
+5 = 19et
+6 = 16et
+7 = 14et
+9 = 11et

+10 = 19-tone non-octave
+11 = 26-tone non-octave

I used sine waves in my experiments.

If anyone uses this VSTi, feel free to verify/correct these results. At
any rate, tweaking it produces some non-12et tunings...
