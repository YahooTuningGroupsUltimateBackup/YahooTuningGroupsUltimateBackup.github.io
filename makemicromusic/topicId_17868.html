All;

For those who may not know, MIDI Yoke is a
utility for Windows that provides virtual
MIDI ports so you can route MIDI streams
between applications.

MIDI Yoke NT (the version that works with
Windows XP and Windows 2000) had always
created 8 ports. For those of you (like me)
using older Windows apps that can only
address up to 10 MIDI devices (like Encore),
this was a real pain, especially if the
device you wanted happened to be alphabetized
after "MIDI Yoke" (like "UA-1000").

At my request, the developer has released
a new version that lets you set the number
of ports. I just upgraded and it works.
You apparently have to install with 8 ports,
reboot, set to 4 (or whatever) and reboot
again. But other than that it's basically
painless.
