Hi there,

Cos(1/x) vibrato starts very fast, in fact theoretically,
it has infinitely many cycles between 0 and any positive value of x.

In practice, you get a very very fast vibrato that rapidly slows down,
and each note on each instrument starts like this.

Anyway here's a 260 Kb clip of a short fractal tune with this effect.
The flute uses tremulo; the rest use pitch variation type vibrato. :-)

http://groups.yahoo.com/group/MakeMicroMusic/files/Robert_Walker/cos_one_over_x_vibrato.mp3
