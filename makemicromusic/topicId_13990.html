Hi,

Is there a program out there that will adapt a just intonation scale to
make the chords sound better? For example (in 12 note JI)- if we have C
= 1/1, F = 4/3, and G = 3/2 then we might have two choices for D... If
we play D with F, we would have D = 10/9. If we play D with G, D = 9/8.
Thus we would need to sometimes play the D with one pitch and sometimes
with another. Is there something out there that adapts this way?

John Berglund
