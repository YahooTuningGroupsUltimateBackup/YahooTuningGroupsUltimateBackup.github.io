Hi all,
I have created a program VLTune for manipulation of tuning table of Yamaha
VL70m sound module.
Using VLTune you can detune individual notes in the octave by -64...+63c
from equal tempered scale, audition results and save/open the tuning table
files (file.det).
You can also create SysEx files with retuning data which can be send to
module from another program.
The program retunes the sound on the "current" position.

Free VLTune 1.1 for Windows is at
www.noise.cz/artech/downloads/vltune11.zip. It is not very much tested, so
any suggestions, bug reports, etc. are welcomed.
