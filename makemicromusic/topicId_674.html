I made a Scala command file for easy calculation
of Horagrams.
The number 3 on the one but last line is the number of
generators in the downward direction and it can be
replaced with another number to get another mode of
the scale. Or use the KEY command to do that.

Manuel

! hora.cmd
echo Create a Wilson Golden Horagram
echo Try out the scale in all keys
echo enter coefficient of phi in numerator
$1:=?
echo enter second term in numerator
$2:=?
echo enter coefficient of phi in denominator
$3:=?
echo enter second term in denominator
$4:=?
calculate/noout 2*(($1+$f plus $2)-($3+$f plus $4))
echo enter number of tones
pythagorean
?
2
0
$0
3
show scale
