Hi Jacky,

Just catching up on some of my MMM e-mail.
I found a neat trick for the Yamaha soft synth,
which has portamento and legato mode, but not
true legato because it triggers the attack
each time.

What I did was to set the portamento on but
set the portamento slide value to 0.

Result is a true legato mode - you get the attack
for the first note of the trill if it doesn't
overlap with a previous note, and then you get
notes without any attack when they do
overlap.

Maybe it will work with other synths too?

As you know, FTS can handle the pitch bends
for monophonic legato trills - or could...

I've just found a newly introduced bug in the FTS beta
that it was switching off the previous note for the monophonic
legato mode - because it noticed the new pitch bend
and so switched off the previous note to avoid
pitch bend resonances, and I'd left out the line
of code to say that it is okay to do the
pitch bends like this for monophonic legato
mode.
