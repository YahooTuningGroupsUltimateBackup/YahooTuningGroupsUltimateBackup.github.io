Todd Booey Chewy wrote:
>Is it possible to load a
>scale into this, do I need to use FTS instead?

If the PSR is XG compatible then it should be possible
to use Scala. But to my knowledge this has only be
done by one person before. First you need to be
sure that your tuning doesn't exceed the +/- 64
cents difference range. Start Megamid and set its
midi output device to the midi interface. Then close it
again. Then in Scala you type

set synth 108
set adjustment
send

send/screen instead of send shows you if the pitches
are within range.
You need to have a recent Scala version, because before XG
tuning had a bug which wasn't found because nobody used it.

Manuel
