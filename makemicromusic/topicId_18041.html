Interesting stuff. They seem to have a 5-note pelog and a 5-note slendro that have three notes in common, which allows them to alternate from one scale to the other while still only using 7 notes. The "Shufflendro" excerpt is different from the others, though; it starts with a different pelog mode and modulates to something that might be a slendro with an extra note in it -- a note that doesn't occur (as far as I can tell) in the other excerpts. I tried playing along with a 23-ET keyboard layout, which seemed like a pretty good (but not exact) fit.
