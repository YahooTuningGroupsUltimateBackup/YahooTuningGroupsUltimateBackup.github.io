Very interesting! But I noted a small typo on this page. You wrote:

Sn = nA1 + n(n-1)
------------
2
That should be, of course:

Sn = nA1 + n(n-1)d/2

That you know this is clear from your rearrangement of this equation
to give:

d = 2(Sn - nA1)
-----------
n(n-1)

So, now I've listened to them, and find them a good set - they
complement each other nicely. Each one evokes quite different
images in my mind. With the third, I couldn't help imagining it
being performed live onstage by a flamboyant pianist, like, say,
Liszt or Grainger.

Now I'm listening to Bill Sethares' latest, and because of that
chance juxtaposition, this thought occurred to me: How would
your "Paintings" series sound if you used a different set of
pigments? More precisely, if you substituted for the present
harmonic piano timbres, a set of timbres whose spectra comes only
from the (non-harmonic) tuning?
