It's entirely algorithmic.

For the scale I used

38.4123 10/9 326.394 100/81 508.798 1000/729 691.202 10000/6561 873.606 9/5 1056.01 2/1

which I made in FTS actually - there's a mean tone scale construction window and
I'd just been working on it recently as it happens.

and the arpeggio (mode):
0 2 4 5 7 9 11 -1

Seed

0 1 0 -1 -2 -1 -1 -2 0 2
- those are just steps up / down in the arpeggio.

I set it with the volume of the seed trailing off from 1 to 0.9,
and I made the volumes recursive so if you have several
layers of seed ornamentation which happen to all have reached
volume 0.9 at the same time, it can get quieter than that.

Also set it so the tune gets a little slower and a little quieter
as it goes up in pitch.

Plus a "polyrhythm" setting of 1 2 4 8 - so not really a
polyrhythm at all, but it changes the tune to call
it one because FTS processes it a bit differently.

That seems to be about it, plus the normal FTS algorithm.
