H-Pi Instruments software updates, released October 3, 2010:

DOWNLOAD: http://www.h-pi.com/downloads.html

- Custom Scale Editor 2.5.0, a free cross-platform tuning editor, now allows unlocking the Live Input window via direct purchase of a license.

- Xentone 1.1.5, a microtonal ear trainer for pitches, intervals, chords, and notation, now includes a new feature: microtonal melodies.

- ScalaVista 1.0.4, a free cross-platform scale browser database, now includes the latest Scala archive.

- Lo-Fi Microsynth (HPLF) 1.0.6, a stand-alone Hz-based, multi-channel additive software synthesizer for Mac, now supports a wider range of USB controllers.

- Plexus Synth Control (PXSC) 1.0.2 is a free cross-platform Roland GS-based editor for Tonal Plexus synthesizers.

- Scordatura 1.0.9, a soundfont synthesizer, notation retuner and control surface design tool for Mac, now includes more example projects.

- Tonal Plexus Editor (TPXE) 1.6.1, a free cross-platform virtual Tonal Plexus, features a new shape design option.

DOWNLOAD: http://www.h-pi.com/downloads.html
