For those of you into DIY stuff, here's how to build a Monome clone with an
Arduino for $140:

http://flipmu.noisepages.com/work/arduinome/

It works out to about $20 for every 4x4 array of keys you put on, plus about
$60 in other parts. The standard model is 8x8 = 64 keys. Doubling this to
make 128 keys would probably be really productive. The keys also have LED's
in them, so you can color them black and white in a Janko/Fokker/etc layout
if you want.
