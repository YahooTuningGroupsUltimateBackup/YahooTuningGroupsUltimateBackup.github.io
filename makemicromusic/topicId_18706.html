200tet can serve as a mean-tone system which approximates both the
harmonic and subharmonic series for the 8th octaves. It is also good
for the Carlos Alpha scale as well as many others. It is also more
simple than 1200tet and can serve as an experimental benchmark when
conducting sound harmonic experiments. Since many modern keyboards can
accommodate almost any 12-note set of almost any temperament then this
idea has a certain practicality which can be implemented in live
performance.
