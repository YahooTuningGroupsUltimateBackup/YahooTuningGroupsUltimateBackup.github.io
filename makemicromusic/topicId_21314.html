It means that development on that code base has
ceased, for a variety of reasons, not the least of
which is that the core objects used in that code
have been deprecated in the IDE used to create it.
This is why the application has been released in its
current limited and unfinished form. It stands as
proof of concept for a future application which
will use a different code base. Download it and
you'll see the idea.
