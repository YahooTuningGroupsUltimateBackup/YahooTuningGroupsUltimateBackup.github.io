Another choice for microtonality in OS X is the Bitheadz Unity
Session software synth/sampler which supports full 128 note MIDI retuning. It's
pretty simple: you type in (for each note) what pitch you want it to play the sample at.
The downside is that there is no obvious way to copy a full microtuning from one
instrument to another. (The workaround is to build the instrument from a template
that has the microtonal scale already built in.)
