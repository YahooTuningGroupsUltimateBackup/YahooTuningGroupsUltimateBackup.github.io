SCORDATURA 0.1.0 Microtonal Software for Mac OSX

New in this version:

- Control Surface enhanced features:
- control duplication, layering and automatic spatial redistribution
- background, grid and off-axis lines with defineable angles, spacing and colors
- arrow key control nudging by pixel or grid spacing
- quick view and transposition of MIDI channels and notes in control groups
- new TextBox control allows captions and new MIDI button elements
- Synthesizer now supports Reverb, Volume and Cents Offset
- MegaStaff notation now shown in the Main and Tuning Table windows
- New example projects are included to show more possibilities
- Help pages have been added for setting up Scordatura with Finale, Sibelius, Logic, and DP
- Numerous bug fixes and improved features
