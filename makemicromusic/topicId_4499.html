as requested previously on this list, AlsaModularSynth 1.5.8 now has full
support for Scala files. The new modules Scala MCV and Scala Quantizer
can
both load scales.

Download ams-1.5.8 from http://alsamodular.sourceforge.net

The Scale Archive can be downloaded from
http://www.xs4all.nl/~huygensf/doc/scales.zip

Use the SCALA_PATH environment variable to point to the location of the
scale files (default: /usr/share/scala).

ams-1.5.8/example_ams_demo_scala.ams is identical to the demo patch by
Bill Allen but it uses a Scala Quantizer with South Pacific Pentatonic
Xylophone tuning.

Have fun !
