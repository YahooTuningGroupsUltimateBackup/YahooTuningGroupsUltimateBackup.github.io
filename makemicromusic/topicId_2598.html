The woefully slow file boxes in Scala have been removed and
replaced with a new one made by myself. Now it appears even
quicker than a Windows file box. On my home pc I measured a
speed increase of about a factor 100 for the archive directory.
It might not have the exact bells-and-whistle-set to satisfy
Joseph, but it will give an info line when a file is selected
of the common types like scales, keyboard mappings, midi files,
etc. You can also have a preview in the main window without
actually loading by pressing the Show button.
http://www.xs4all.nl/~huygensf/software/Scala_Setup.exe

I've added MIDI relaying to Scala now. There are already a handful
of programs that can do that, but I thought it would be practical
to have it integrated in Scala. It's quite basic, monotimbral, and
without the advanced features that Fractal Tune Smithy has. I have
tested it with a MIDI keyboard and with loopbacked MIDI files.
You can change the scale while it's relaying, like in FTS.
The feature is in the Tools menu.
Additionally, if you open the chromatic clavier, you can view the
notes there. (Is that of use to anyone?) You can even use it
simultaneously, playing a duet with MIDI keyboard and mouse. If
you use a keyboard mapping, you can for example have quarter comma
meantone on the keyboard, and 31-tET on the screen.

http://www.xs4all.nl/~huygensf/software/Scala_Setup.exe
