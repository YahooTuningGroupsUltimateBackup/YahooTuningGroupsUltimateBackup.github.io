I have used (in order of satisfaction)
http://pages.google.com is quick, free, and relatively
privacy-invasion free, especially if you already have an email
account. 100Mb is enough to start.
http://tripod.lycos.com is kludgey but it works
http://myspace.com only lets you have 4 files
http://soundclick.com has more file space, but it requires
registration for downloads.
http://zebox.com only lets you have 3 (unless you pay) and makes the
listener jump through more hoops than it should, like Soundclick.
