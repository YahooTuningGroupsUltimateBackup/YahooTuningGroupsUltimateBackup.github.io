Also, somebody made an interesting discovery concerning the custom
mtof (MIDI-to-frequency) Max/MSP object generated by this software.
The idea is to use plug this object into a Max/MSP synth patch to
retrofit it for the tuning specifically encapsulated into this
object. The discovery was that this object also works with synths
external to Max/MSP, including the free Apple DLS Synth included with
Mac OS X.
