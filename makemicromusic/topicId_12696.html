19-TET + JI work:

My interactive work "Sand" is on the web at
http://music.columbia.edu/~chris/sand
Previously, there were just MP3s up on this page, but now the full
interactive version of the piece is up, for public download.

Briefly, for "Sand" I wrote a rather intensely detailed composition, and
then created an interface so that instead of having to force yourself to
listen to it from beginning to end, you can "surf" the piece, and/or view it from different perspectives, as if it were a painting or a piece of sculpture. (You can read more about this on the site).

Although it's written in MAX/MSP, you don't need the application to run it. You can download a "standalone", or else use it with the "RunTime" (free) version of the application. You'll need a broadband/DSL connection to download, as the files are pretty big. Also, it definitely works on Mac OS X, and a random duude in Italy assured me that he got it to work on PC.
