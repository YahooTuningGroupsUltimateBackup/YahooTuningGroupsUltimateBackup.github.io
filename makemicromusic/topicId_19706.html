Another musical algorithm for making microtonal transformations is:

C = zero cents
G = x
D = 2x
A = 3x

E = y
B = x + y
F# = 2x + y
C# = 3x + y

Ab = 2y
Eb = x + 2y
Bb = 2x + 2y
F = 3x + 2y

Choose any values for x and y.

Subtract multiples of 1200 when the results exceed 1200(an octave).
