>Will there be a table to fill in or do I need
>another line prompt command to build a table or file to send?

No, SEND will create the midi file, or if you want to
transmit the midi file by hand you can do instead
SEND/FILE xg.mid
Then use a midi file player to send it. The default way
is to invoke the Megamid player to do this, but the
first time you have to set Megamid up to use the external
midi port for output.
