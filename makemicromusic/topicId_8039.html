List,

The recent version of Sonar came with the TTS-1 softsynth from Roland. Lots of useable things in there for people locked into 12tet, but there are a few sounds I'd like to liberate. First and foremost is a cool pizz acoustic bass. Naturally, the synth doesn't accept non-12 tuning.

It looks like I'll have to go through this laborious process:

1. Record a midi part with a bass sound on a tuneable synth (so I can hear what I want, pitch-wise)

2. Save that track as a midi file

3. Open Scala, set the tuning I want, import the file, retune it, save as new midi file.

4. Load new midi file into Sonar, load up the Roland bass, make sure it sounds right, export it to an audio file.

5. Dump the bass midi tracks, import the audio file as the new bass track.

Yeah, no one said it would be easy. But I'm wondering if anyone out there can come up with a way to streamline this; I'd love any suggestions. If you're curious about the sound of the bass, I did a quick-and-dirty groove:

http://www.microtonal.org/mp3/groove_temp2.mp3 (~0.5mb)
