I understand that there are easily found kraks available for the (now discontinued) Logic Audio EMagic 5 on PC platform.

This is the ultimate midi/sampling music production program (now on OSX as 6.4.2),, and also great for video sync etc.

All versions can read and play pitchbent midi files.
EXS, its software sampler, can be remapped for any microtuning system to play to the nearest cent.

The obsolete PC version would also run VST plug-ins, but uses the dreaded ASIO for audio.
