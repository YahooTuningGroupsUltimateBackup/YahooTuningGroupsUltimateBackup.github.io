FM7 on Mac OS X can load .syx files and also receive MTS messages.
I've had no problems using Max Magic Microtuner to export tunings
to .syx and have FM7 read the .syx files. It can run both standalone
and as a plugin (Audio Unit, and I think VST too).

Caution: FM7 on Mac OS X has a stuck-notes issue which is
intermittent. I reported it to Native Instruments in December. I
got an email from them saying they were working on it, but no update
since. I gotta wonder if my Nord Modular G2 will have both the DX7
emulator _and_ microtuning implemented by Clavia before NI finishes
fixing this issue.
