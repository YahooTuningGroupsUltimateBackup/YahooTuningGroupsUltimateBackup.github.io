I've been testing this, and have come to the following conclusions:

1. It has phenomenal quality at the standard setting.

2. It doesn't give great compression at the standard setting.

3. The developers are dangerous maniacs. The encoder overwrites the
wav files without even bothering to ask, an insane thing to do for a
lossy encoder and something which should never be the default for any
encoder.
What is wrong with these people??

4. It is very badly supported, but the WinAmp plugin does work and of
course in Windows you can associated the mpc extension to WinAmp.

5. It is, therefore, not yet ready for prime time, but worth keeping
an eye on. If I want to overwrite a wav file for now, I'll use FLAC,
thank you very much Mr. Loony-Tune programmers.
