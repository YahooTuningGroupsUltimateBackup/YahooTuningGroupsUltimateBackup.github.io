The rules are user-friendly and can be customized. Select
or (de-select) the rules you want and see what you get. This
software is particularly valuable when it comes to using external
midi devices (eg Kurzweil) which can be tuned to any 12note sets of any
temperament and in any note order. Thus the original notes are
known, the pathway can be set, and the destination notes can be heard
in real life. A scientifically controlled musical experiment is the
result. Check out the manual which is on-line for more details. I am
still very new to this software.
