I don't use Scala to make scales for Absynth -- there may be a way to do it, but I seem to remember that Scala just exports a text file that you need to copy and paste into the Absynth scale editor (on the MIDI tab) one note at a time. I use the OpenOffice spreadsheet to calculate pitches in semitones for each note in the scale, then copy and paste notes into Absynth. Since I use scales with a period of repetition (usually an octave or a fraction of an octave), I can set the "Keys per 8ve" and "8ve interval" to the appropriate size for the scale I'm using and only need to copy the notes for a single period of the scale.
