I've got a new 3'52'' work up at akjmusic.com

It's called 'prism'. You can access it directly at
http://www.akjmusic.com/audio/prism.mp3

or from the 'works' portal page at http://www.akjmusic.com/works

It's a mutitracked synthesizer improvisation using Kraig Grady's Centaur
scale, which I tend to use by default when I want to write in 7-limit JI
because it seems to be best for that purpose ;)

There's a bit of a neo-Terry Riley feel I think, with the digital delay and
all, sounds sort of 'Shri Camel'-ish. And a bit of Morton Subotnick
FM-squawcking for added texture going on. Hope you like it!
