Hi Dan,

I expect you've been answered by now, but if not,
I've just discovered that if you download and install
the mp3-tech site version of the LAME mp3 encoder
(which is a good free command line encoder)
it also automatically installs RazorLame, which
is a windows front end for it which is also very good
and free.

Download url:
http://www.mp3-tech.org/software/encoders/lamewin32.exe

from:
http://www.mp3-tech.org site
see:
http://www.mp3-tech.org/encoders_win.html

It's just a matter of downloading the setup
file and running it, and you will find the encoder
under Start | Programs | Lame | RazorLame.

Just choose File | Add files for all the .wav files you
want to convert, and then Actions | Encode, and leave
it to get on with it.

Using Edit | Lame Options you can set the bitrate
and various advanced mp3 options.
