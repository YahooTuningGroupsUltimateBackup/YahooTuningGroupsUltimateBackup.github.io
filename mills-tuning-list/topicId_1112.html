One nice way of using dynamic just intonation in performance is
to use a set of organ bass pedals that retune the synth to the key center
of the pedal currently being pressed. It seems to me I read somewhere
that Wendy Carlos uses a left hand keyboard for something like this, but
with the bass pedals both hands remain free.
