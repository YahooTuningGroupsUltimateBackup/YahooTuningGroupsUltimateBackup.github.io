Kevin,

> I'd like some pointers to web sites, books, articles, etc. that
> might get me up to speed on the nomenclature you're using here
> as well as the basic premises of JI, experimental tunings, etc.
A good place to start is
ftp://ella.mills.edu/ccm/tuning/papers/bib.html
At the bottom there is a set of links. You can visit the Just
Intonation Network and John Starrett's website which also provide
many links for further exploration. The above bibliography also
contains links to articles on the WWW.

> I've done
> quite a bit of conventional composing and arranging and up to
> now have relied on scalar "modes" to convey various "flavors".

ftp://ella.mills.edu/ccm/tuning/papers/modename.html
is a big list of modes.

> If you could direct me to some representative
> recordings, I'd be very grateful.

ftp://ella.mills.edu/ccm/tuning/papers/discs.html
is a list of microtonal recordings with many kinds of styles.
