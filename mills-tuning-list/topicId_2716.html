I've recently been playing with the program that Greg Phillips wrote for
PCs with a SoundBlaster card. It was written primarily to assist in tuning
wind chimes, but I have found it very useful for many other uses. As well
as generating reference pitches with reasonable accuracy, it converts
ratios to cents and vice versa, reads you own scales from a simple text
file (as well as the data files for Scala) input as cents or ratios, it can
play random notes from a scale, or hold a chord as you adjust each pitch,
making it geat for trying out different temperaments. Requires Windows 3.1
(may work with '95, but I don't know for certain), a SoundBlaster or
compatible and very little else (I tried it on a 386 with 4 Meg RAM and it
worked fine).

It can be downloaded from:

ftp://ftp.maths.bath.ac.uk/pub/dream/xen/chime.zip

Usual commercial disclaimers apply.

-- Pat Missin.
