As discussed here some months before, the new .TUN format specification
is finished and released.

Detailed specifications, free tools and free source code is available
here:

http://www.mark-henning.de <http://www.mark-henning.de>

What's new:

* Informations about the scale included, e.g. scale
description (needed for database or search applications)

* Scale construction using simple yet capable algorithmic
functions

* Multiple Scales per file

* Keyboard mapping

* Scales can be embedded in other files, e.g. HTML without
loss of compatibility (means: you can read the .HTML file
directly like you do with .TUN files)

* ...
