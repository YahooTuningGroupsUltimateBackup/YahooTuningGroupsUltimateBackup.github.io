Web page-based javascript program that automatically generates beautiful scales

http://www.geocities.com/djtrancendance/PHI/generatesymmetric.html

This will generate beautiful scales from any number between 1 and 2 that divides the space between 1 and 2 in highly symmetric ways.

And, yes, that includes being able to generate a scale from the perfect 5th IE 3/2 (just enter 1.5 into the program to do this). Obviously 3/2 is special because it divides the space between 1 and 2 evenly (50% on each side).

For the 5th you can derive the basic scale:
1
1.13541666
1.29
1.5
...and, using the optional "period equivalents" you can obtain the "extended" scale (which nears 9 notes per 2/1 octave):
1
1.06684
1.1354
1.21875
1.291666
1.5

Side note: this program uses the number entered as the period/"pseudo-octave".

Hope you all enjoy this...please let me know what you think.
