Tonal Plexus Editor TPXE 0.1 (beta release) is now available
for Mac (Universal binary). It is hidden from the H-Pi website
but can be downloaded here.

<http://www.h-pi.com/software/TPXE-MAC.zip>

This software facilitates the creation of tuning tables for the Tonal
Plexus. Even without having the actual keyboard it lets you get a
feel for what it is like to work with the keyboard.

The minimum screen resolution requirement is 1024 x 768.
Installation may require that you drag the Ariell font into your
fonts folder. Provisional documentation is included. Your input
is appreciated; minor changes may be made at any time. Please
send comments, feature requests or bug reports.

NOTE: The Windows version will follow shortly, and at that time
both versions of TPXE will be announced on the website and made
available from the downloads page.
