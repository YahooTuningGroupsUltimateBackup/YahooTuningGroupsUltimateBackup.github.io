ok,

See if this is what you want under starling

http://xenharmonic.wikispaces.com/Comma+pump+examples

It looks to me that the pump has 20 4 note chords.
So the procedure is this (assuming a vertical time line)

1. click edit this page
2. place cursor where you want the table
3. go back up the page and click table
4. number of columns is the maximum number of members in the largest chord.
5. number of rows is the number of chords
6. place cursor in first box
7. press space (or any number of spaces you desire)
8. enter ratio
9. press tab to go to the next box
10. press space
11. enter ratio
12. press tab - note that the tab key will automatically wrap around to the
next row when you complete a row.
----
if you have a blank box (lets say a 2 member chord where the largest is 4
members) I suggest you use a space as a place holder.

I would like to mention that you could put these into Excel (or similar
program), take a screen shot and paste that into a graphics program.
After a bit of editing like cropping the picture you can embed the picture
formatted as you like in whatever program is easiest for you to do this.

I hope this helps,

Chris
