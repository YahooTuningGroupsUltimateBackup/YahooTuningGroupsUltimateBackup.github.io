Caleb asked: "I suppose that a completely consistent fingering (5ths all the same number of steps) is mathematically impossible with a 5th at 703.06613 with 27 steps?"

The answer is that this is definitely not a mathematical impossibility, as this is in fact a fine tuning for Misty[27], the 27 note MOS scale for misty temperament, which tempers out the misty comma, |26 -12 -3>. 800 cents minus the above can be used as a generator.
