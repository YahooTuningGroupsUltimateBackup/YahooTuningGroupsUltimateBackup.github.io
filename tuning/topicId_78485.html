TPXE 1.0 for Mac and Windows is released today.

This is scale creation and editing software for Tonal Plexus keyboards, fully functional and free. Try it out as a virtual version of the keyboard. The new Mac version now loads SoundFonts (.sf2) - makes any soundfont microtonal.
