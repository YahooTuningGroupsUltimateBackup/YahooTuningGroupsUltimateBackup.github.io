Tonescape was written for Windows and only runs in Windows.

Ozan Yarman (a member here) has been trying for a few years
to get it running on a Mac under Parallels Desktop, and he
posted the other day to the MakeMicroMusic list that after
updating to PD7, Tonescape now works perfectly:

http://launch.groups.yahoo.com/group/MakeMicroMusic/message/28205

There is no "true OSX version", and probably never will be.

I have succeeded in getting Tonescape to run under Linux in a
Windows virtual machine (running under qemu), but it is
too slow for actual use ... i'm still working on that.
