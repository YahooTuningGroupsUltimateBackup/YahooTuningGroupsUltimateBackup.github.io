Hi, everybody! :)

I'm a 54-year-old Scottish/Welsh guy currently living in Indio, CA, USA, and have finally gotten fed up with the 12TET tuning system.

That resulted in an 11-tone symmetrical superscale that I'm anxious to try out, and I was wondering if anybody had heard of this particular scale being mentioned anywhere else before.

I also need advice on alternate tuning freeware and/or shareware for Windows 98 using a SoundBlaster 16PCI sound card.
