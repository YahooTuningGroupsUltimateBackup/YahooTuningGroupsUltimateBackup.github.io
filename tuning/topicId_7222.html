There's more info on the list at this web site:

http://nowitzky.hypermart.net/tuning

It looks like you already got yourself signed up. The list is virtually
unmoderated, so you're free to post whatever (hopefully relevant) info or
questions.

And you might want to use the above web site to search the list's archives
for your topic of interest.

By the way, I've been by the selftuning.com site before. Looks like an
interesting instrument! The tuning list has many guitarists who I'm sure
would have an interest in discussing The Performer guitar.

The tuning list often talks about "microtonal" issues, as opposed to
typical alternate guitar tunings. The list tends to focus on the "notes
between the cracks" of the conventional equal-tempered system. So as far
as guitars go, they've got some weird-looking frets on some of 'em. For
example:

http://nowitzky.hypermart.net/justint/jiguitar.htm

But alternate guitar tunings are within the scope, so post away! Hope you
find what you're looking for...
