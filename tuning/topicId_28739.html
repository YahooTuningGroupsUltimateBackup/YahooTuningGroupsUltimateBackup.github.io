Lately I've heard about prime-limit and now thanks to the dictionary of tuning terms by Joseph Monzo and others I know exactly what it is.

And now I know that I've used it often using the term "Prime Intervals" ( for six years ), then I didn't have an internet account.

I've read also in the tuning dictionary that "Some theorists attach special importance to the prime numbers as they seem to each produce a unique affect in music".

I have the same opinion and I can tell clearly that all 5-limit scales ( which have only 5-limit and less-limit intervals ) are joyful,
and every 7-limit scale ( which has some 7-limit intervals ) has some sadness.

If you want to try this make sure not to temper-out the 255:224. Use something like 171 Et, if your device allows you why not using cents.
