Other notes on version 1.5.4 - Changes to the MTS checksum calculation

The MTS checksum calculation rule used in the MTS file import and
export functions for the MTS Bulk Tuning Dump message has been amended
to conform to the checksum calculation rule used for the MTS Key-Based
Tuning Dump. Please read the MIDI Tutorial documentation file enclosed
with the Max Magic Microtuner package for more information on this topic.
