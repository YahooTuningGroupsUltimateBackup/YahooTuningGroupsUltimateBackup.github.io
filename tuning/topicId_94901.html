The current keyboard discussion is very nice! But it should be moved to Make Micro Music, for it is a discussion about practical musical instruments. Were someone to bring up string spacing and pitch order for lyres, the topic would immediately be tagged as more suitable for MMM, and this keyboard discussion is no different.

Also, the harmonic entropy discussions need to go to the Metatuning list, or an appropriate psychoacoustics list, for they're about psychoacoustics.

I bring these things up because, pressed for time, I come here to catch up on responding to a whole slew of posts about tuning issues, and find these interesting, but off-topic, threads have flooded the site.

Discussion of keyboards for alternative tunings is not off-topic here. Several years ago there was an extended discussion of keyboards for pajara on MMM, and Jon Szanto raised the issue that it could be construed to be off-topic (since it didn't involve any actual music), but he let it go on, because the objective was to build a keyboard that would be used to make microtonal music.

This list was here before MMM, so just about anything related in any way to alternative tunings is on-topic here, even links to new compositions (normally found on MMM). The exception would be threads involving higher maths, for which tuning-math was specifically created, so as not to put off folks who don't appreciate that sort of discussion.
