CSE stands for Custom Scale Editor, a free microtonal scale creation utility for standard MIDI keyboards (no TBX1 required), and TPXE stands for Tonal Plexus Editor, a free microtonal scale editor for Tonal Plexus keyboards (no Tonal Plexus required). The Mac versions make any soundfont microtonal.

These are bug fix updates mostly for mac PPC users, but they also contain minor improvements and a new feature.
