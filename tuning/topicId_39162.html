This replaces the text concerning copying text in the SCALA window to
the clipboard (Thanks, Manuel, I was doing it backwards! :) )

If you want to save your window text, so you can open it in a text
editor or wordprocessor, you can use another routine written for
SCALA by Robert Walker.

First enter

@C

in the the SCALA command-line box at bottom.

Then enter

SHOW, so there's something that will be copied.

Finally, enter in

@COPY

Now, the text with your tuning information is in your Windows
clipboard, so all you have to do is go EDIT PASTE (or Ctrl-V) to
paste it into MS Word, the Tuning List postings, or whatever.
