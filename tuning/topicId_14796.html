Paul:

While in your IE browser,

- click on the 'Tools' menu
- select 'Internet Options'
- click on the 'Advanced' tab
- scroll down to the 'Microsoft VM' heading
- check 'JIT compiler for virtual machine enabled (requires restart)'

That should do it for you.

If these choices don't appear in your 'Advanced' tab, then you'll
have to re-install Internet Explorer as Carl suggested, and make
sure you include the Java (which Microsoft calls 'VM') options.
