Francesco,

I would recommend the yamaha tx81z synth module.

The yamaha tx81z synth module is:

1. Very inexpensive - commonly available for $60 to $140 on ebay.
2. Easily retuned with a computer (tuning is stored in memory for a gig).
Any midinote can be tuned to any pitch. Also has octave based retuning.
3. 1.56 cent resolution
4. You can load patches on it to expand the sound bank
5. There are folks on the list that can help you get started with it.
Relative quick learning curve

The pitch resolution is not the best but most synthesizers are either 1 cent
or 1.56 cent. For most purposes this is good enuf.

The only way to retune a SB soundcard is with pitchbend.

I would suggest you get the program SCALA for dos. Very good scale
generator, analyzer, and synth retuner. I use it to retune the tx81z.

For realtime softsynths try mellosoftron, which is a realtime sample player.
If your samples are microtuned, then you can build up alt-tuned scales.

I use midikeys (www.shareware.com ) and VMCI-virtual midi control
interface -- to configure my computer keyboard as a music keyboard and then
send output thru the joystick port on the SB card to the retuned tx81z.
midikeys is cheaper but VMCI is much better ($85 US).

Darren Burgess
