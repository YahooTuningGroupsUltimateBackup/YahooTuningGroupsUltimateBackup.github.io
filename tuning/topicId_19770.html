I've just added a new database

FAQ questions from newbies

Idea is, if you have a question needing an answer, you add it here.

It will then appear in the tree under
FAQ | Questions needing an answer
next time the tree is refreshed.

When it is answered, the answer will be put in
FAQ | Answers to questions from Newbies.

For the first month after the answer, it will
also appear at the end of the relevant section
in the questions needing an answer page.

Just an idea again, but I think it has a good
chance of working well.

Pat, I wonder, can you add your solution to the tree?

- to do so, save it in the faq folder in the files
area as a text file.

Or of course, upload it to your own web site,
and use the url for that one (see previous post).

Then, add it to the database giving the url
to your new files/faq file in the url field, and
Calculations as the parent folder (assuming you
want it to go there).

To add it to the answers database, you look for
the original question, and fill in the
Folder for answer | Entry that gives answer
fields as:
Calculations | How do I convert a ratio to Herz.

Ask if you have any questions, or if this isn't
explained clearly.
