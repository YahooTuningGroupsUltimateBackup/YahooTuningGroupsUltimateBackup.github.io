The JiCalc hypercard stack for the Mac can be used for any type of
tuning, not just JI. For ET's, one simply enters a ratio (which can be
calculated by the program) for 1 step, then define the "octave" as one
degree. This duplicates the interval for the whole midi gamut. Only 64
notes can be indicated per card, but multiple cards can be defined for
large, non-repetitive tunings.

to the Onelist site when I get my primary computer repaired, hopefully
early next week.
