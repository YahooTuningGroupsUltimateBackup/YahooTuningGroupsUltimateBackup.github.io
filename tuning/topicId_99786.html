#1. First and foremost, I don't like doublewide temperament but I just want to demonstrate the symmetry within the chord progression (major-minor, minor-major, major-minor, minor-major). In the doublewide example, the equivalence interval is 2/1 and the period is 600 cents. In the meantone example, the equivalence interval is 4/1 and the period is 2/1.

#2. FYI, both examples were created using the same sequence file, only the actual scales were different.
