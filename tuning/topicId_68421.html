Hi - Graham, I got the .csd file to work. It's great, thanks a lot! I'm going to modify it a little and post again.

To anyone else who had trouble running Graham's file: I installed Csound on Windows to the default location (C:\Program Files\Csound), then moved Graham's file to Csound\bin (naming it with a .csd extension, e.g. "breed.csd"). Then at the command prompt I navigated to the correct folder (cd C:\progra~1\csound\bin) and typed the command "csound breed.csd". The default operation writes the output to the soundcard rather than to a file. Sound doesn't start until you grab one of the sliders and move it.
