There was a lot of discussion here about month ago on the
topic of virtual pitch. That is something that I have been

http://home.austin.rr.com/jmjensen/VirtualPitch.html

I downloaded C source code, written by Ernest Terhardt, from
Richard Parncutt's website, converted it to Java and added some
graphics and a user interface to allow you to put in your
pitches in terms of 12-eq note names, or JI fractions.
