OSX recommendations follow the links:

Cameleon.
http://www.camelaudio.com/cameleon5000.php?flash=1

Melodyne. new version 3 coming soon. (free demos - amazing application)
http://www.celemony.com/melodyne/

Max Microtuner.
http://www.hitsquad.com/smm/programs/MaxMagicMicrotuner/

Spear. (audio analysis and synthesis - very good freeware )
http://www.klingbeil.com/spear/

Logic Pro 7 - muchos dinaros but well worth it ;-)
http://www.apple.com/logicpro/

Make VST plug-ins work with AU Core Audio
http://www.fxpansion.com/product-auadapter-main.php

Audacity: (free player for ogg etc.)
http://audacity.sourceforge.net/

VersionTracker for OSX - searchable database and links for new OSX audio applications etc. + useful user reviews
http://www.versiontracker.com/macosx/

Limewire (freeware) for peer to peer. (kraks galore;)
http://www.limewire.com/english/content/home.shtml

How I am microtuning using OSX.
http://www.lucytune.com/midi_and_keyboard/pitch_bend.html
