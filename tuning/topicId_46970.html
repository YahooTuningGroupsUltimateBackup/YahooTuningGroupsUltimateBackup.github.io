Michael,

Handel works well with a variety of Baroue temperaments. I think 1/4 comma
meantone only works and sounds wel for Handel in isolated passages...

You might want to try out any of the following: Silbermann 1/6 comma,
Werckmeister, Kirnberger, Valotti-Young.

For a list of cents offset values from 12-et, check out:
http://www.millersrus.com/avery/

Hope this helps....please post your results to us so we can hear...sounds fun!

>Also, can anyone tell me how to correctly calculate the checksum for a
>Roland SysEx message?

Add all the byte values starting with the part id until the last
one before the checksum. Then
Checksum := (128 - (Bytesum mod 128)) mod 128
