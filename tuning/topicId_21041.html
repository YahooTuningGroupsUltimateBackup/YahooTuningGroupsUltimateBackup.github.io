I haven't seen this mentioned so here goes. There's a new software
synthesizer for Windows with some nice tuning features, such as unlimited
number of scales, full MIDI note range retuning, interval entry, dynamic
retuning, and resolution to 0.000000000001 cent. It is called Midicode
Synthesizer and the URL is http://www.midicode.com . It has also been added
to my web site.
