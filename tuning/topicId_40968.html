Alexandros,

> Are there any requirements for a synth in order to be retunable by
> Scala?
> I must know , in order to ask the manufacturer about exact MIDI
> specifications.

Yes it should have an internal tuning table which can be changed
with system exclusive messages. If that's the case you should send
me the message format so I can the implementation to Scala.
The V5 looks like a cute device, but I think it's unlikely that it
supports microtuning, however you can ask.
If not, and it supports pitch bends (should ask too), then you
can use midi-relaying via Scala or Fractal Tune Smithy, but this
imposes some restrictions.
