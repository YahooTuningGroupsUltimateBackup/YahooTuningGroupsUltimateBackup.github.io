Hi All,

I found this neato Macintosh program for making chord charts called
Chord Studio 1.0 by Greg Chapman. URLs below.

A few points to note :
1. Can be used for Xenharmonics since double clicking on a chord
symbol on the chart lets you change chord names to
any text characters.
2. Uses Quicktime (or Midimanager) to play a chord,
but QT can be routed to your favourite synth.
3. Just use a sequencer or editor, etc. to send tuning messages to
your synth, so the chords you play in Chord Studio will be "in tune"
4. You can add a song title, author etc.
5. It is non-disabled UK�15 shareware.
6. You can unfortunately only have one time signature per chart.
