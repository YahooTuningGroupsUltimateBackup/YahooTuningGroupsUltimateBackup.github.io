Sorry, there isn't anything. Kurzweil sells the Expressionmate MIDI ribbon
controller, however, it cannot generate note-on/note-off simultaneously with
pitch bends. You have to define one portion of the ribbon to generate
notes, then define another portion to generate pitch bends, if that is
acceptable.

Rick,

Hello!

I almost forgot to mention to you that there is a fairly new and
interesting product from Korg called the "KAOSS Pad", which allows
you to route a number of simultaneous realtime controllers to a
module at once and control it all with you finger tips on this small
pad that you slide about on. It looks really cool. Seen them for
around $250.00. The only thing I'm unsure about is if it generates
midi-note ons or not. I'd like to get one of these myself - I love
using unusual controllers - I've been a Midi Wind Synthesist for many
years. One of my fantasy midi controllers is the Mallet Kat - if only
money were no object!
