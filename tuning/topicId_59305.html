--- In tuning@yahoogroups.com, Kraig Grady <kraiggrady@a...> wrote:
> <>Hi Gene!
> Could you define epimorphic again.

Suppose you have a JI scale, numbered in order of interval size above
the unison, so that the unison is 0, and going to n-1, where n is the
number of scale elements in an octave. Then it is epimorphic with val
h if h(s[i]) = i, where s[i] is the ith scale element.
