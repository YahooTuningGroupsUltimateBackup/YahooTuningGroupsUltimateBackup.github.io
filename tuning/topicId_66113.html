We belive in R N-EDO = 2^(M/N) for N-EDO system

Then M/N = log(R N-EDO) / log(2) = CN-EDO /1200 which C stand for cent.

New R' A-EDO of N-ADO system is

2*(M/N) = 2* (log(R N-EDO) / log(2)) = CN-EDO /600

That is :

R'A-EDO = CN-EDO /600
