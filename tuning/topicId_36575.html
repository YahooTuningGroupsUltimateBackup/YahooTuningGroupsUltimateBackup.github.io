Uncork the champagne! After many years with requests for a
Mac version of Scala, now there finally is one.
Thanks to Tobias Giesen, the latest console version 1.82
was ported to MacOS X. At the moment, this is an alpha
test version, and we're asking for people to test it
thoroughly. The download location is

http://www.tgtools.de/ScalaX_1.82_alpha1.sit

Instructions:
- unstuff, move to a convenient location
- go into Terminal window, use the cd command
in order to go into the folder where the
unstuffed files are,
- start Scala using the command:
./scala
- send bugs and comments to Manuel.

Tobias was mentioned here before as author of Finale plugins and
tools. More than a year ago, we were trying to make a version for
the earlier MacOS 9 after Gary Morrison and I bought a toolset which
was required for the compiler to run. But it was too cumbersome and
Tobias got stuck with the finish in sight. Now with MacOS X, it
became very easy and required only small adaptations, which however
still needs to be proven by testing.

Who knows a simple MIDI-file player for OS X? At the moment we're
looking for one, which is necessary for the SEND command to work.

Also looking for people who can provide programming experience
for the MIDI-out port under OS X.

When circumstances permit, we'll try to port Scala 2.x to OS X too,
but that depends on the port of Gimp and its toolkits to OS X, and
may take a year or so.

Homepage: http://www.xs4all.nl/~huygensf/scala
Scale archive: http://www.xs4all.nl/~huygensf/doc/scales.zip
