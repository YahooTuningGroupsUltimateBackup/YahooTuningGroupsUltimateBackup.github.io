I've placed an update of Scala, version 2.02 at
http://www.xs4all.nl/~huygensf/software/scala20win.zip

It has some minor improvements, bug fixes, a few extra
menu items, some more notation systems and it can now display
different plots (i.e. graphs). There is a plot for temperaments
showing the deviations of fifths and thirds along the cycle of
fifths. Also a plot of the current attribute can be made, so
for example it's now very easy to make a graph of harmonic
entropy for example. Also one can choose between line and bar
graphs.
There is no Linux version yet, I can probably get to that in
November.
