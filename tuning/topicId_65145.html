Version 1.5.5 - Release Notes

- new feature: direct intonation table cell editing via Command-click
- new ALL SOURCES mode in the MIDI Settings window
- new ALL DESTINATIONS mode in the MTS SysEx window
- added documentation: Scale Building Tutorial for starters
- bug fix: the Reset button now actually transposes the keymap back to
zero when in Shift Table mode
- free and immediate upgrade for licensees of previous versions with
the enclosed Self-Updater tool
