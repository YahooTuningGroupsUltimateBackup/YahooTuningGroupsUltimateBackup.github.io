Sure sounds familiar. My first MIDI stuff was done on a
board with similar resources but was based on 6802. Two
quite elaborate algorithmic pieces were entered via hex
keypad. One of these pieces "Polyphonic Variations" still
works too 'cause I burnt an EPROM (the noise of an FSK
cassette is like going to the dentist!) 6802 architecture is
the same but has different size registers. The public-domain
cross assembler I use on the MIDI Tool Box (68HC11) might
allow disassembly of 6502. If you find the code and would
like it dissassembled, send me the file.
