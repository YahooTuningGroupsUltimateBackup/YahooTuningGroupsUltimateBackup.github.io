For those of you who might be interested, I have updated the
Microtuner software (OS X) with a feature that makes it compatible
with Absynth. Here are some additional details:

The new version (OS X only) can be downloaded at the following
website:

http://groups.yahoo.com/group/16tone

Direct download link:

http://digilander.libero.it/microtuner/Microtuner147GLY.sit

Version 1.4.7 - Release Notes

- new export format compatible with the Native Instruments Absynth
software synthesizer

- the enhanced Multiplier tool now includes a standard
intervals "palette" for easier operation

- enhanced intonation table editing features now include dragging
and dropping a frequency value within a Microtuner window ("move"
function), and between two different Microtuner windows ("copy and
paste" function)

- MIDI note on events are used to highlight the corresponding
intonation table rows in real time

- a "Panic" button is now available in the MIDI Settings window to
send an "all notes off" message to the QuickTime synth

- various bug fixes

- free upgrade for users of previous versions, including multiple
license owners
