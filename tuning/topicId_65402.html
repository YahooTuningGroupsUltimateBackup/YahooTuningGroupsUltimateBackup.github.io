Since in 12EDO every interval class can be sounded
from every pitch class in either the ascending or descending
direction, I think the initial 13-note all-interval-class/all-contour
melodies should be created/generated in 12EDO and then
translated/transposed/transported into other tuning systems
which have the same number of pitch classes as the chosen
melody.

Approximately 1/3 to 1/2 of the melodies I've generated so
far embody a 5-tone subscale, which is amazingly economic
and efficient to me. With a 13-note melody, that means that
there are 8 additional notes that will determine the subtlety
of the pitch-class/scale-step centricity/heirarchizaion of
each melody.

The resulting pitch class collections would be from 5 through
10 tones, from what I've experienced so far, but there might be
11 and 12-tone melodies out there to be discovered yet.

These "reference melodies" would give one way to compare
various tunings with the same number of pitch classes.
