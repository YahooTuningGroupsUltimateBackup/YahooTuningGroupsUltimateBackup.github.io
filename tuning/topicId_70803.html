New Microtuning Tool for Nord Modular G2

paolovalladolid <phv40@hotmail.com>

3/30/2007 7:52:12 AM

I didn't buy this synth for its tuning capability (there isn't any
inherently), but the ability to translate Scala files into
specially-tuned patches for the G2 has just been added, courtesy of an
external software application suite, called "g2ools", written by
members of the G2 user community. So if you own one of these synths
or are thinking of picking one up, here's what you do:

1. Download and install g2ools. Download location and instructions
for installation are here:

http://electro-music.com/forum/viewtopic.php?t=15405

2. Download the new Scala-loading-and-conversion tool from here and
follow the instructions:
http://electro-music.com/forum/viewtopic.php?t=10387&start=25
(Look for G2SCL.zip)

The forum thread in #2 above also has posted sample patches tuned to
Fokker and Meantone. The author says he generated those off of Scala
files using the new tool.

3. Visit the electro-music.com forum mentioned above to give feedback.
