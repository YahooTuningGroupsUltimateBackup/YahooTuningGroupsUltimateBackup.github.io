Scala can turn a .seq file into midi very quickly. This means that a
media player or file renderer could use a .seq file as a media
format, bypassing midi entirely, while using many of the same
definitions. The fact that it is ascii would be a plus.

The hard part, of course, would be convincing people who write
software to actually do it.
