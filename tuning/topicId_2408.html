Thanks to Dan Stearns I've realised why some folk have been having problems
with my Harmonic Complexity measures spreadsheet. The LCM and GCD functions
are not available in the standard installation of Excel 97. If you see
nothing but #NAME? in some columns try this (from the help file):

Run the Setup program to install the Analysis ToolPak. After you install
the Analysis ToolPak, you must enable it by using the Add-Ins command on
the Tools menu.

How to Install the Analysis ToolPak

If the Data Analysis command is not on the Tools menu, you need to install
the Analysis ToolPak in Microsoft Excel.

To install the Analysis ToolPak

1 On the Tools menu, click Add-Ins.

If Analysis ToolPak is not listed in the Add-Ins dialog box, click Browse
and locate the drive, folder name, and file name for the Analysis ToolPak
add-in, Analys32.xll�� usually located in the Library\Analysis folder�� or
run the Setup program if it isn't installed.

2 Select the Analysis ToolPak check box.

Note Add-ins you select in the Add-Ins dialog box remain active until you
remove them.

Good luck.
