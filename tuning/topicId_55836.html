Yes. You got it!

Or putting it another way, 155 cents is closer to 150 than it is to
167.

Instead of using the degrees of 72-EDO, we could use the actual
cents corresponding to the ratios. Using the formula

cents = 1200 * log(large_part_of_ratio/small_part_of_ratio) / log(2)

To one decimal place
11:12 is 150.6 cents (neutral second)
10:11 is 165.0 cents (wide neutral second)

155 cents is closer to 150.6 than it is to 165.0, so we get the same
result, that it is a neutral second.

Consider if you had been asking instead about a 170 cent interval.
This falls clearly in the category of "wide neutral second", however
if your scale (or the set of scales you are considering) did not
have any intervals that fall in the (plain) "neutral second"
category, then you might choose to drop the term "wide" and simply
refer to the 170 cent interval as a "neutral second".

This sort of flexibility is what allows this system to remain
consistent with the common usage in 12-EDO, where for instance 400
cents is simply called a major third.
