Here's yet another special purpose meantone. Here the fifth is set to be
exactly 356525/512, or 696 173/512, cents. This is between the fifths
for 31 and 50, near that of 81. The feature of this fifth is that it
is flat from 700 cents by 1875/512 cents, which multiplied by
8192/1200 results in 25. Hence the fifth is 700 cents, pitch bended
down by 25, and any of the pitches in this system will have pitch
bends of 8192 plus or minus 25n, for integers n. Hence if a midi file
in this tuning is converted to a text file or examined in a midi
editor, it is easier to tell what we are looking at, which can be
useful if you are doing retuning.
