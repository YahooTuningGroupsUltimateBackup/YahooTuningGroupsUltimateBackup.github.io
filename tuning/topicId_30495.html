Paul,

Crap. I couldn't sleep last night, so I did some searching. The
printer driver I use was a promotional from Adobe that came with
Adobe Type Manager, but I wanted to check more...

Did some searching and found a number of 3rd-party options, the most
promising of which I came up with was PDF-Xchange:

http://www.docu-track.com/pdf-xchange.htm

I downloaded a demo (you must supply a valid email for them to send a
password for the demo). It handled multiple typefaces and graphics
quite well, and seems to have a lot of options for the final output.
It may not have all of the options that the Adobe product itself
might have, but it appears that it would suit your purposes (and
anyone else here who would want to publish portable documents with
graphics and text). This software is only $29.95, much less than
Adobe Acrobat, Writer, or Distiller. Please note that this is a
Windows product; if you want to investigate more resources or for
other OS's, search download.com (or any other online file repository)
for "pdf"...

HTH,
Jon
