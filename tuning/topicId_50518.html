Scala doesn't send midi to the synth directly in order to tune it.
Instead, it creates a midi file and needs a midi file player to send the
data. If the installation went right, you would see Megamid appear on
the screen when you do @send. But Megamid's output device needs to be set
first (click the setup button). You can also start Megamid from Scala
with the command "spawn megamid".
