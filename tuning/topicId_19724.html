Hi John (JdL)

To put your new database entry into a sub-folder do this:

In the folder field put the name you want for its folder
e.g.
Adaptive tuning

That is all you need to do to create a new top level folder
in the new system.

If instead you want to put that folder inside another one,
add a new entry.

If you wanted it in one of the folders already in the tree
you'd add a line

Calculations | Adaptive tuning
or
Notations | Adaptive tuning

To put it in a new top level folder,
use

New folder name | Adaptive tuning

Leave the url field blank, and in fact, you can leave all
the other fields blank when making a new folder.

See how I did it for

Calculations Cents to from ratios

and compare with the tree

I've updated the table headings as I realised they weren't
clearly expressed.

Second column now is
Short descr, (entry) or sub folder (to make new folder)

Third column now is
Url - leave blank when using to make a new folder
