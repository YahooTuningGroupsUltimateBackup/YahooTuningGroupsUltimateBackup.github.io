For the version with them uncommented, remove the c in e.g. cdekany3_1_3_5_7_11.wrl
from the url.

- can modify to make any other dekany by replacing same number everywhere in the batch
file to make the .wrl file from the template.

To change the title that you see, edit the .wrl file and look for the line

string "Dekany"

in the first Billboard node.

N.B. as picture of 4D figure, all the faces, including all the triangles, and all the tetrahedra and octahedra
are external (in the same way that all the squares, lines and points are on the outside of a cube) -
there is no internal structure. If one can imagine that. I wonder how many people really are able to
see 4D shapes in 4D.

I imagine rather few, though some have been reported to be able to visualise
4D shapes and answer questions about them by doing so - don't know if that is the
same thing.
