KVR isn't exactly a hotbed of xenharmonic activity, but if you do a
search there on tuning you will find some threads about softsynths
and .TUN support.

Mark Henning did a thorough job with the documentation and
free coding tools for the expanded ,TUN and .MSF formats,
including some really clever things such as HTML embedding
and algorithmic scale coding within the file. The documentation
PDF is worth a read for tuning enthusiasts, and I really hope
synth developers give it the attention it deserves.
