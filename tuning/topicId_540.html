Graham asked...

> Tell me how you can be sure I didn't take an old MP3
> file and revise the date to make it look new.

Any time you move a file to a new location (someone else's machine across
the net), the date gets set to the date the new copy was created. Unless you
take some special action to re-set it to an OLD date (which isn't always
easy on any operating system), the date is new. When I download a file, it
gets set on my disk to the date that I downloaded it. Using file dates
between systems isn't reliable at all, and you can't tell a thing by the
date.
