I'm a bit nervous about everyone downloading sound files and
graphics each day along with the digest.

One alternative (which might not be too much work?) would be to
have someone host a website with "free" upload access. Then,
whenever someone wants to "send" a picture or soundfile along
with a message, he/she could send along a http pointer to the upload
site - this way the reader could click on the pointer to
hear the sound or see the picture if desired, but the file
wouldn't be downloaded unless wanted.

In terms of maintenance of the site, the host would need to go
on and delete all files greater than (say) 3 days old.
This way space wouldn't be too much of a problem. If the
host was clever, this deletion could probably made automatic.
