This is done with the Ethno 2 package as an exercise in learning how to use it.

This a an arrangement of For Flute and Piano put into the Ethno2 Celtic sound set of Guitars in E and D and Penny flute.
I might have over done the grace notes a bit... but there you go.

Both versions use a micrtonal tuning that comes with the package. The appalachian tuning has a E flat that necessitated some movement of pitches and still has a rough patch 0:48 - 0:52 but I still prefer this tuning.

http://notonlymusic.com/board/download/file.php?id=177
