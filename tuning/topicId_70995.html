For microtonal capable synths, what if a different Tuning Box,
call it TBX-MTS were available that employed any and all existing
MTS Sysex. Would anyone be interested?

Although microtonal capable synths don't require something to be
hooked up in order to make microtonal music, they do require
programming and so on.

The advantages over relying on a computer running software with
a MIDI interface would be the portablility and general ease of use
you normally get from a dedicated hardware device. As this would
be a new product, we could build it so that the entire Scala archive
could exist in memory with room to spare. It could also interface
directly with Scala, if Manuel collaborates.

What do you think? If you like the idea, are there any other functions
you would like to see in such a product?
