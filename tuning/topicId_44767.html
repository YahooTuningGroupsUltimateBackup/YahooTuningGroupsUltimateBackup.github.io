Max Magic Microtuner version 1.3.3

Release notes - New features:

- increased Scala compatibility: with the new "Export Scala"
option you can now export native Scala .scl files that are fully
compatible with Scala for Windows;

- "Scale Inspector" tool for displaying scale intervals both in
terms of cent deviations and ratios;

- Undo function.

Program description:

Max Magic Microtuner is a Macintosh application for OS X and OS
9 (Carbon) that allows Max/MSP and Pluggo users to generate
their own alternate tuning external objects by converting specially
formatted text files into Max/MSP MIDI-to-frequency externals.
These tuning text files contain the intonation table data and can
be created and edited with Max Magic Microtuner itself or with any
other text editor: the program will read data from both plain text
files and native "mttx" type files. It can export the intonation
table
data in a text format that is compatible with the read method of
the Max "Coll" object. It can also import and export native Scala
(.scl) microtuning files.
