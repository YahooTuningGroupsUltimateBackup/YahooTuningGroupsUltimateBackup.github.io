Patrick Pagano asked for examples of my Csound work. Musical examples
can be found at http://www.mp3.com/PrentRodgers and the material that
produced the sound is described at http://pws.prserv.net/music1 . I use
a pre-processor to create the scores. The preprocessor, called
samples.exe is available at the web site. It takes a primative score
language as input and writes a Csound .sco file. This is combined with
the orchestra file (also on my personal web page) by Csound to create a
.wav file. MP3 reduced versions are posted on MP3.com. The music
explores the tonality diamond in a variety of ways.
