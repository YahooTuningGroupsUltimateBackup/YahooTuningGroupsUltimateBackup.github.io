Hmm...

FM7 receives standard MIDI tuning bulk dump. That's number 107 in the
edit/options/midi menu. Select save synth tuning from file menu. That
will save the tuning as a .mid file (.sys not needed). Then import it
into FM7 as sysex. Should work. I just created a 17-equal tuning and
it worked perfectly.

Do you have the update from Native Instruments website?
