These are free microtonal scale creation utilities. CSE = Custom Scale Editor, for standard MIDI keyboards (no TBX1 required), and TPXE = Tonal Plexus Editor, for Tonal Plexus keyboards (no Tonal Plexus required). The Mac versions make any soundfont microtonal.

These updates contain bug fixes, improvements, and new features.
