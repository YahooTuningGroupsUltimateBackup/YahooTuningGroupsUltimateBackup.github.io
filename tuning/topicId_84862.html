Here's a simple scales based on one of my calculators I created for Ratioed Just Intonation scales.

Western Scale is the 12ET scale or the 2-12ETJI0,0 or 2-12RJI0,1:1:1:etc scale. (1:1:1 isn't needed, it's actually 2-12RJI0,1 scale, but showing the difference in annotation)

This is my Phi-7RJI3,Phi:Tri:Qua:Qui scale with the sequence of ACBDACB. I uploaded it into the WhistlingElk folder.

Phi is the 2-nacci / Fibonacci's ratio of Xn/Xn-1, or the golden ratio.
Tri is the 3-nacci / Tribonacci's ratio.
Qua is the 4-nacci ratio.
Qui is the 5-nacci ratio.

A:B:C:D

I start all my scales off with concert A 440hz.

440.000000 - B0 / Start
467.739369 - A1
503.076924 - C1
539.282478 - B1
580.866972 - D1
617.487162 - A2
664.138113 - C2
711.934955 - B2 / Phitave

A Cent 105.841608059380 A% 1.06304402061003
B Cent 120.314572628101 B% 1.07196822605064
C Cent 126.088982393288 C% 1.07554967040311
D Cent 128.599970195202 D% 1.07711078318158

A% means multiply previous frequency by that and you get an A step.

The file goes

B0,A1,C1,A1,D1,B1,C1,B1,
C2,A2,D1,A2,B2,A2,B1,A1,
B2B1B0,C2C1,A2A1,C2C1,B2B1,C2B1,A2C1,D1A1,
B1B0,B1A1,D1C1,A2B1,C2D1,B2A2,D1C1,B1A1,
B0,A1,C1,B1,D1,A2,C2,B2,
C2,B2,C2,A2,C2,A2,D1,B1,
C1,A1,B0,A1

13 measures in 4/4 at 72bps.

Yes, you are absolutely correct, but harrisonj only plays 440Hz if you set it to A, so you may have overall tuning problems with other keys.

The tunings on the lucytune site are intended to be used in key of A, so that they will then match with the tunings of other LucyTuned instruments using A=440 Hz, e.g. guitars, other DAW's etc.

So set the key in the microtuning window to A, and it should all work perfectly.
