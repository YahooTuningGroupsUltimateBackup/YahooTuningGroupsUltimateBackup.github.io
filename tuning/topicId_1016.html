> Is there a method where I can easily convert a scl or
> something created fm modes to sysex message
> without turning to windows or something else
> e.g. *.scl for synth 2 or 107
> easily converted to according sysex message????

The best you can do is the following:
Create a file called dump.cmd with these two lines
ECHO Give midi-file name:
SEND/FILE ?
The question mark is important. Then start Scala and do
SET SYNTH 2
ITERATE @dump *.scl
Now for each file you must type a name, you can type over
the original name from the screen without extension.
It's a useful suggestion to make it automatic altogether
but that's not possible at the moment.
