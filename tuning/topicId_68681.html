Java Microtuning Organ

1/5/2007 7:12:04 AM

I just released a new version of my Java Microtuning Organ (JMTOrgan
v1.01) that allows you to experience playing on keyboard instruments
in historical temperaments and pitches using soundfonts such as jeux
1.4 (or other instrument samples). The program relays output from a
midi-enabled keyboard to a soundfont enabled soundcard (e.g.,
soundblaster audigy) and retunes in the meantime. The program is meant
for practical use during rehearsal or performance: you can set up a
stop board of presets that you want to use in a performance and cycle
through them easily by pressing the spacebar.

The prorgam can be downloaded for free at http://www.jedosoft.info.

Jeroen Donkers
