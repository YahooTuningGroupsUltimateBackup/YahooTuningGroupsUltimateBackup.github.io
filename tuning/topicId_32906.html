Glad you like it, thanks.

I wonder if you know that it is customisable too. Easiest change
- if you have a program with gradient fill such as PaintshopPro,
prob. most of the paint packages, you can get the same
effects, but with other colours, by editing the files
in the folder Images-Original and saving them on top
of the ones in the folder Images. I used
the various types of gradient fill such as radial gradient
and sunburst gradient in Paintshop Pro.

The readme in the Images folder says a bit about this.

N.B. I found that when one exits from it,
sometimes it will crash with a "this program has done
something illegal and will be terminated" type bug.

I.e. you click Exit, and at that point Windows
tells you that it will terminate the program.
It is exiting anyway, so not such a big thing,
not like the one you found in autumn 2000.

It was an easy fix too once I discovered it.
It was because of the new buttons, because
it was trying to update them when the window
they were in had vanished already, or was about
to go away, or something like that.
