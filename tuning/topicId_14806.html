Many thanks to all those who pulled out all the stops to help make this a
reality. Notably Keenan Pepper, Oscar Lanzi III (who responded to my post
in sci.math) and Paul Erlich.

Oscar expressed the required transformation matrix as

M = (I + f(n)*E)/sqrt(2)

where
I is the 3x3 identity matrix
E is the 3x3 matrix consisting entirely of 1's
n is the number of dimensions
f(n) = (sqrt(n+1)-1)/n

I used Keenan Pepper's solution since it was already unpacked for 5D. I
haven't checked if Oscar Lanzi's comes out the same.

About the tumbling eikosany. It is certainly fascinating to watch and
should run for several hours before repeating, but I'm not sure I'm any the
wiser about eikosanies for watching it. If you see the donut, be sure to
record the 4 angles and tell us.

Someone else might want to modify it to add all the other edges as Paul
Erlich suggested, or try a dekany in 4D.
