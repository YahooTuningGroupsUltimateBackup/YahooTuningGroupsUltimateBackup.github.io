Dear Chris,

In the script editor of each instrument (check the manual where that is) you can select pre-defined tuning scripts like "Dynamic Pure Tuning", "Microtuning" and the like (although the latest version of the Kontakt library contained some bugs in these scripts).

There is the Kontakt Microtuner script.

http://www.12equalboresme.com/Scala2Kontakt/index.html

Also, please find my extended version of the predefined Microtuning script attached. I added support for detuning via CC messages on the fly (CC 1000-111 change pitch classes 0-11 in cent steps), changed the script to also detune release samples, and it can now be combined with other "tuning scripts" such as the Solo Instrument Performance Suite (SIPS) which supports legato and expressive vibrato (put my script behind those).

Check out the documentation how to install scripts...

Best
Torsten
