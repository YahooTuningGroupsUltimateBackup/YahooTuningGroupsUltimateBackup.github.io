Hello tuning list,

Quintet.net is a network performance environment originally designed
for performances on the Net and/or in local networks.

It features real-time notation and can notate microtonal music on five
grand staves in eighth-tone precision.

It also has a polyphonic sampler built-in with 1 cent resolution as
well as VSTi and MIDI playback (with pitch-bending).

You can play on your MIDI keyboard, your computer keyboard or use a
table to send sequences of pitches in arbitrary tuning.

Each tuning can be filtered through another tuning. For instance, an
equidistant pentatonic scale can be filtered through (approximated to)
a Bohlen-Pierce scale.

Quintet.net can import an unlimited number of tuning files, which can
either be created manually as text files or created automatically with
Victor Cerullo's Magic Microtuner utility (which in turn can import
tunings from a library with >1000 tuning files in mtx format).

Quintet.net is built on Max and requires a Max installation with the
free (!) MaxMSP Runtime application (Macintosh or Windows).

I'd like to invite you to download the program from
http://www.quintet-net.org/dl-frames.html and to test-drive it.

If you're not sure and want to find out more about it, please download
the Getting Started document first, or go to www.quintet.net for
further information.

Please respond off-list, if possible.
