Hi Jon,

You can stream mp3s using the m3u format

An m3u is just a text file like this:

science_friction_short_extract.m3u
=
"
http://homepage.ntlworld.com/robertwalker/science_friction_short_extract.mp3
"

I.e.the m3u is a text file consisting of a single line
which gives the link to the mp3.

Example to try:

http://tunesmithy.co.uk/science_friction_short_extract.m3u
(modems)

or
http://tunesmithy.co.uk/science_friction_short_extract_128.m3u
(broadband)
