(see link: http://www.xs4all.nl/~huygensf/scala/ ) which also contains a
MIDI file picker "MegaMid".

The limited number of simultaneous notes is caused by the fixed
amount of MIDI channels in the MIDI standard, so it's not a limitation
of Megamid as such.
The MIDI Tuning Standard is supported by a few softsynths like
Timidity (free), Native Instruments Absynth 2.0, FM7, Pro-52 (commercial).
Audio Compositor's implementation is unfortunately broken.
There are also some hardware synths that support it, see the Scala
documentation.
