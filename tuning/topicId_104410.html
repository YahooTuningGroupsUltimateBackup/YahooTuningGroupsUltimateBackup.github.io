This is a random tuning that comes up every so often in XA chat, but
the name for which hasn't been posted here. It's a bit higher error
and inconsistent, which means only some will like it. It's in 16-EDO
and is exactly this

http://x31eq.com/cgi-bin/rt.cgi?ets=5p_16p&limit=2_3_5_7_9_11

It's called "slendroid" because it's a play on the words "slendro" and
"android," which is because this is right between slendric and
machine. You still get that three 8/7's gets you to 3/2, but you also
get in machine that two 8/7's is 9/7. It's good for marimba sounds and
so on.
