I had to do some maintenance (euphemism for debugging) on
EXAMPLE/MIDI, the Scala command to retune a MIDI file.
So it's a warning if you use this command in Scala 2.0
and earlier.
As one of the tests, I've put Telemann's famous recorder
suite in A minor in 1/4-comma meantone. I like it better
that way! The file is
http://www.xs4all.nl/~huygensf/midi/telemann-suite-mt4.mid
No webpage. Copy this file if you want to keep it, because
I'll remove it after a while. If someone hears a flaw let me
know. Thanks,

Manuel
