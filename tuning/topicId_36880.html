Not much to say, I'm getting better on my 31et G&L
although some would say my current pursuits are
rather pedestrian (playing bebop). I've discovered
lots of nice scales on a daily basis that sometimes
do (and often don't) have anything to do with something
I can interpret in either music or tuning theory.

As an example, here is a nice two octave scale

4 4 5 4 4 5 4 4 5 4 4 2 4 4 5 ( 4 4 )
0 4 8 13 17 21 26 30 3 8 12 16 18 22 26 0 ( 4 8 ).

One can ponder it for years on paper but if you write
it in tablature for a normal guitar tuning it is

E x | | | | x | | | | x |
B x | | | | x | | | | x |
G x | | | | x | | | | x |
D x | | | | x | | | | x |
A x | | | | x | | | | x |
E x | | | | x | | | | x |

...which is really easy to finger and play
(fast, too) AND sounds good for all kinds of things
including those mbira patterns that African guitarists
play.

Give it spin,

Bob Valentine
