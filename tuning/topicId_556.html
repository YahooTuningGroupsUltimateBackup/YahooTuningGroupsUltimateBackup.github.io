Hi Mary,

You should be able to relay any scale no matter what number
of notes per octave, non octave, etc. in FTS.

One can set up any keyboard mapping one likes from the midi
in note numbers to the notes to play in the current mode
and scale.

There are a few defaults available as presets such as white
notes -> mode, black notes -> mode (useful for pentatonic
scales), all notes -> mode, every second white note -> mode.

Any in between notes map to accidentals, i.e. + or - one
or more notes in the underlying scale.

Do ask if there is something to be figured out - I expect
the help (what there is of it so far) isn't clear.
