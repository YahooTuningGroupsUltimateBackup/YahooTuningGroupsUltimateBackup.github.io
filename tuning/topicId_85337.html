Scordatura 0.3.0 licenses available

Aaron Andrew Hunt <aaronhunt@...>

Licenses are now available for SCORDATURA for Mac OSX, unique software which retunes and reroutes MIDI output between any virtual and physical MIDI ports, with any number of tuning tables, and also allows design of custom MIDI control surfaces using standard elements like switches, buttons, trackpads, joysticks, mod wheels, knobs, user definable polygons, etc.

DETAILS: <http://www.h-pi.com/SCRDsoftware.html>
DOWNLOAD: <http://www.h-pi.com/downloads.html>

New in version 0.3.0
- OSX 10.6 compatible
- Numerous bug fixes and improved features
- Licenses can now be purchased for just 53 USD
