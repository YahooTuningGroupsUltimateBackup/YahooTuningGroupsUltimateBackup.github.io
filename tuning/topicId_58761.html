Multiple tunings using Cameleon 5000 with Logic - also available for M$ software.

5/25/2005 6:04:48 PM

I have been experimenting with Cameleon 5000 in Logic Pro 7.1 on Tiger 10.4.1, and found that it is possible to have two or more Cameleon plug-ins running simultaneously
It is therefore possible to have a different tuning on each track, so that you can use more than twelve notes per octave live, or with existing unmodified midi files.
Cameleon does not seem to be effected by changes in the Song Setting tunings, which control the overall tuning of all the Apple instruments. Cameleon plays the tunings which have been loaded into each track of the plug-in.

I have put up 49 Cameleon microtunings (about 35 new ones) linked from by the Cameleon image at:

http://www.lucytune.com/midi_and_keyboard/pitch_bend.html

They can be downloaded in .zip or .sit formats, and include instructions etc.

Have fun! Experiment! Enjoy!
