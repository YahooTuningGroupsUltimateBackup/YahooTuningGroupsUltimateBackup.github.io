I have Linux at home now, and made Scala version 2.03 for Linux.
The user interface looks and performs better than under Windows.
And you can dress it with "themes", which is not possible under
Windows. Only a few improvements over that version, there's a
cycle plot now (View->Plots->Cycle plot) which is a graphical
reproduction of the "show cycle" output, and the Help window is
improved.
Download from
http://www.xs4all.nl/~huygensf/software/scala-20-pc-linux.tar.gz
