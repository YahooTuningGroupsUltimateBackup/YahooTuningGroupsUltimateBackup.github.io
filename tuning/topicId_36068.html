I just finished writing up a detailed, step-by-step web page on how to use
Yamaha's "Expert Editor" to tune VL70-m sounds to arbitrary microtonal
tunings (or at least a tuning that can work with a single
completely-arbitrary tuning table). Without this technique, it only allows
for 12-tone-per-octave tables wherein the pitches within each octave only
vary from 12TET by no more than 64 cents. Feel free to check it out:

http://www.lonestar.texas.net/~mr88cet/retuningVl70m/microTuningVl70.html
