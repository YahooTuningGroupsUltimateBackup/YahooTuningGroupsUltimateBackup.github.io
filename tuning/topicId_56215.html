(Microtuning just got simple with Logic 7)

I have added more LucyTuned microtuning files, for use with Logic 7,
There are now 50 files, including a full readme, for you to experiment with.
Put them in your Tuning Tables folder at: Library/Logic/Tuning Tables/
Select the tunings from: File/Song Settings/Tuning

You can now apply these tunings to your existing tracks or new compositions to experience the difference that LucyTuning can make to your music.
All files are all based on A = 440Hz. and use conventional Western notenames, familiar to meantone users.
Arranged in this way makes all these tunings entirely compatible with all other LucyTuned instruments, which are all set from the reference pitch of 440Hz.
This system even works with vocals and audio autotune - check the readme for details.

Free download from:

http://www.lucytune.com/downloads/lucy4logic7.sit

Enjoy!

More details on Logic 7 microtuning are at:
