There are such keyboards. They're usually based on 12 note octaves, so if you want more than 12 notes you need to be careful what you buy. The most flexible thing is to buy a standard MIDI keyboard, plug it into your computer, and run one of the many fully retunably soft synths.

Retuneable hardware synths / keyboards:
http://www.microtonal-synthesis.com/micro_af.html (click on the other letters as well on the op of the page for the rest of the alphabetical list)

This page features another list of both hardware synths and software synths / samplers etc.
Also this is software that will allow you to retune standard synths / keyboards etc by realtime midi pitch bend messages (though this can be a bit of a hassle and uses up a new midi channel for each note of polyphony and this doesn't work with everything)
http://www.huygens-fokker.org/scala/
