Do you want this pattern to be modulable on demand into any key, or do
you want to function strictly modally within these pitch relationships
(i.e., the F key always sounds the same pitch)?

If you any key always plays the same pitch, then you'll find a lot of
possibilities, from Ensoniq to Emu to Kurzweil. If you want it to be able
to be modulated arbitrarily on demand, then you won't find as many
alternatives.
