Johnny Reinhard asked me to repost the info for the show tomorrow.

For anyone in the NYC area, tune into WYNC-FM 93.9
tomorrow, Friday May 18th at 11pm.

For anyone outside the area, WYNC streams on the asx protocol.
Which means you can listen to it with Windows Media Player,
but you can NOT listen to it through the QuickTime Player.

WNYC-FM 93.9 streams through:
http://www.wnyc.org/audioindex/wnycfm.asx
which you can type, or copy and paste,
into WMP's "Open URL:" dialog.

If you don't have Windows Media Player, you can download it at:
http://www.microsoft.com/windows/windowsmedia/en/download/default.asp
If you've never used it before,
you might want to give yourself some time to set it up.
