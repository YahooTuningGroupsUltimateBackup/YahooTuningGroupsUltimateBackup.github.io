Yes, besides the Dd and Ad you could improve the intonation of the
other tones too. It's easy with my program Scala, you type the following:

mode/equal huseyni
set notation T24 (the Arel-Ezgi system)
show
@smithy

To see which makams are built in, type:

mode/all makam

You need to set up the program Fractal Tune Smithy to do what
you want, relay the messages from the MIDI-in port to the
soundcard for example. It uses pitch bend messages so you
don't need to change soundfonts.

First install Scala:
http://www.xs4all.nl/~huygensf/scala
and then FTS:
http://www.robertwalker.f9.co.uk/fts_download.htm
Follow the instructions in the file fts_and_scala.html
