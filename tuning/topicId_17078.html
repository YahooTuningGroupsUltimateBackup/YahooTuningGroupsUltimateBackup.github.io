I think that a genre, in music, must indicate a whole set of stylistic preferences, which microtuning is included in. So I think that microtuning itself is not a genre. Consider the "Rock" genre. Microtonal rock is possible in addition to 12tet rock; the microtuning only changes a specific part of the whole.
