I've been using the xg soundcard and YES it does support microtuning.
However, it only has the repeating octave (12 tones/octave) kind of
tuning. But it is possible to send it new tuning tables within a piece.
One way of doing that is with the Justonic Pitch Pallet. Another way
would be to embed the sysex within a MIDI file.
