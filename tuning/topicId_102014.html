Suppose S is a set of JI intervals i including 1 and 2 with 1 <= i <= 2 such that if i is in S, so is 2/i. S is intended to represent a set of pitch classes defining "consonance". A JI interval c is an essential tempering comma for S if:

1. c is greater than 1 but less than the smallest interval between any two members of S

2. There are three intervals i, j, and k in S such that c = i*j/k

I think that this is a fantastic idea. It might also be useful to look
at the case where c = (i/j)/(j/k) = i/j * k/j, which then creates a
single chroma turning k into j and j into i. A good example of this is
875/864, which has the property that 7/6 -> 6/5 -> 5/4 -> 9/7 are all
separated by the same chroma. Another good thing to look at is the
case where c = (i/j) / (k/l), which simply tempers out the difference
between the chromata of two disparate sets of intervals, regardless of
whether or not they form any sort of "chain."
