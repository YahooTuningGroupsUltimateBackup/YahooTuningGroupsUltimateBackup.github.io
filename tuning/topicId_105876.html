Dear friends,

I want to give you some information I found/derived recently which
seems to be important. This matter deals with well known data and it
might have been discussed before. The twelve exponents of a constant
"@" to form a scale and the curious six coincidences of six scale
tones as well as the round cell numbers 100, 150, 300, 350, 450, 600
make uncommon features of this subject. @^12 has the appearance of an
abnormal scale octave that complies with the following relations:

2/@^12 = Pythagorean Comma = 1.01364326477.....

@ = (1.58740105223 / 1.5) = 1.05826736816

Other relations can be applied to get 1.58740105223.

At least the first five decimal frequency figures of the remaining six
cells, which are numbered 50, 200, 250, 400, 500, 550. We can forget
the cells. By checking the mentioned scale which can be call it "the
exponential tones scale", we have:

@ = 1.05826736816 ..............Cell # 50
@^2 = 1.11992 982212...........Cell # 100
@^3 = 1.185185....exact.........Cell # 150
@^4 = 1.25424 2806485..........Cell # 200
@^5 = 1.32732 423362............Cell # 250
@^6 = 1.40466392316 exact....Cell # 300
@^7 = 1.48650999285 ,, 1.48651 exact..Cell # 350
@^8 = 1.57312 501761............Cell # 400
@^9 = 1.66478687188 exact....Cell # 450
@^10 = 1.76178 962115..........Cell # 500
@^11 = 1.86444 444...............Cell # 550
@^12 = 1.97308073703 ..exact..Cell # 600

(2/1.97308073703) = 1.01364326477 = P. C.

It is clear that I only found these strange coincidences.The set is a
reduced octave scale. In my opinion, somebody should do an analysis of
this new system, perhaps nothing is new.
