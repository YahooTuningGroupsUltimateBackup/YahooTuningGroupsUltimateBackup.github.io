Hi Flynn,

If no one here give you any info, you might try posting at Making
Microtonal Music:

http://launch.groups.yahoo.com/group/MakeMicroMusic/

... and offshoot of this list. There is at least one member there that
uses Finale with notation tweaks and pitch bends, Rick McGowan. Bear
in mind that this is *not* simplistic stuff right now, as Finale isn't
built from the ground up for microtonal work. You might also need to
upgrade to a new version, not sure.

Let me see if I still have a link to his page...

Ah, yes:
http://rm-and-jo.laughingsquid.org/FTS-HowTo/MicroOrchestra.html

The big drag in your case is Rick is on a PC, and part of the solution
is using an external program to relay the midi signals to a sound
source, tuning along the way - and all the apps he tried/uses are PC
based. Maybe Mac people will know.

The other option, on a Mac, would be to consider Logic, which Charles
Lucy uses (on this list) successfully. I don't know if the lite
version has this capability, but worth checking out.

OK, that's all from someone who neither owns a Mac or uses Finale!

Cheers,
Jon

Hi Flynn,

The only thing i have to say about this is that trying to
do any kind of just-intonation in Finale is a nightmare,
because tuning adjustments are based on deviations from
12-edo. It would be much easier to define a tuning which
is some type of equal-temperament.

If you can wait (i wish i could say how long), the current
version of Tonescape will be released as a free demo in the
somewhat-near future. Composing in JI in Tonescape is as
easy as blinking your eyes.

You can get a used desktop Windows machine good enough to
run Tonescape for about $200 or maybe even less.

You certainly won't need the upcoming Windows Vista,
because Tonescape is designed to run optimally on Windows XP.
It also runs fine on Windows 2000 (which is what i have on
my own system), and in one case has been cajoled into running
on Windows 98.

I don't know if Tonescape will run on the Mac version of
Windows, because it makes heavy use of Microsoft's proprietary
.net and DirectX features. But i really don't know anything
about Mac ... and since i've finally just recently made the
leap into Linux (and love it), i doubt if i'll ever bother
with Mac.

And no, we don't have any plans for a Linux version of
Tonescape any time soon. Probably eventually, but it's
a long way off.
