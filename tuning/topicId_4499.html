Hi Folks,

I have placed my C language computer program to assist with tone hole
placement (for flutes and related woodwinds) based on equations given by
Benade and Hopkin on my music webpage at

http://cjchapm.home.mindspring.com/music.htm

(look under "Flute, Bansuri")

I put the code in a WinZip'd file that includes the C source code, two
example input files, and a Win32 console executable (for Windows 9x,
etc.). If anyone is interested in the program but has problems with
WinZip files and/or web access, let me know and we can investigate
alternative delivery methods. :-)

BTW, I've released the code free under the GNU General Public License.
If anyone has suggestions for improvement, finds errors in the code,
etc., let me know and I will be happy to update it. I compiled the C
source under Microsoft's C compiler, but it's a text-based program, so
it should be easy to port it to other compilers and operating systems.
