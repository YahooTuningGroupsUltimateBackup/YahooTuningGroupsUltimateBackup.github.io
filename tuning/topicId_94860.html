When the number of notes to the scale is small, the number of temperaments associated to it can really pile up. Below I list a matrix M for the period-generator mapping for some 7-limit temperaments, followed by another generator mapping to the keys of a hex button lattice keyboard. The rows for these are can be obtained by matrix multiplication by [4 1]M and [3 1]M, corresponding to the 1/4-1/3 7-note MOS region. Next I list 7-limit JI intervals which when tempered give the tunings for the button generators. Note that despite all of the varied 4 and 3 button mappings, none correspond to the patent val mapping. As usual, you may want to set fixed width fonts.
