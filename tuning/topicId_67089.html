Hello people.

I just uploaded the latest version of microabc (2006-06-28), a free (GNU GPLed) software to help making microtonal music with ABC tools (abcm2ps, abcpp and abcMIDI).

The new version includes a basic built-in preprocessor (just a macros replacer) which can be used instead of abcpp in some cases, as well as a Sagittal-2.0d8 PostScript font (for microtone accidentals). This is not yet the final version of Sagittal-2 font, it is released for testing purposes and can be subjected to changes (thanks to Dave Keenan and George Secor).

A relatively complete support for Sagittal Notation System is being worked and it will be available soon (maybe in next release). Just now, you can use abcpp and the files `sagittal.fmt', `sagittal-pfb.fmt', `sagittal-mixed.fmt', `sagittal-midi.abh' and `sagittal-midi-12et.abh' to write music in Sagittal notation. See inside `examples/sagittal/'.

Any comments are very welcome.

Best wishes.

[[ I have got a message from Jack Campin, who is also interested in to use microabc for Turkish music. (I have not any MacOS-X binaries, but I suppose it is easy to compile.) ]]

However I am not familiar with Maqam. It seems that the first microtonal ABC program (a patch for jcabc2ps) was written to do Maqam music notation. It is limited to quarter-tones. The legal accidentals are (using the nominal C):

Standard ABC:
__C ^^C (double-flat and double-sharp)
_C ^C (flat and sharp)
=C (natural)

jcabc2ps quarter-tone accidentals:
_/C ^/C (half-flat and half-sharp)

Afterwards, a more general microtonal support was implemented in abcm2ps (users can create their own accidental glyphs). Next, such accidentals were accept in abc2midi. The "/" is used to represent a semitone fraction (1/2 by default):

^31/100C (C raised by 31 cents)
_14/100E (E lowered by 14 cents)
^1/3C (C raised by 1/3 of semitone)

Thus, if you want only quarter-tone accidentals, there is no need to use microabc; abcm2ps and abc2midi can do that. (If you need special accidental glyphs, please send me samples.)

However, if you want to use different tunings, microabc will be useful. (You can, for instance, get irregular scales mapped from the usual/quarter-tone accidentals.)

What do you need specifically? What will be that Maqam interface for microabc?
