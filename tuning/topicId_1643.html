The much awaited Scala version 1.5 for Windows 9x/NT is now online,
the location is:
ftp://ella.mills.edu/ccm/tuning/software/pc/scala/scala15win.zip

Scala is a very complete freeware editor/librarian/analysis tool for
tunings. It allows scales to be created, manipulated and combined in
many different ways. A large library of scales is available for it. It
can tune various different synthesizers and samplers via standard
MIDI-files. To install, unpack scala.zip in an empty directory and
read the readme.txt file.
After startup, HELP shows the commands, and @HELP creates a window
with the help text.

Version 1.5 is an update with over 10% more functionality than 1.4.
This release does not run under DOS, only Windows9x/NT. This means
the 8+3 file name limitation is removed.
If people still want a DOS version then e-mail me.
A few minor bugs were fixed. The DIVIDE and FIT commands run faster.

Some of its new features:

- The lattice of a rational scale can be shown in triangular form.
- The command EQUALTEMP/DATA shows the consistency levels and Paul
Hahn's diameter values and more other things.
- Many new note name systems were added for the command SET NOTATION.
- FIT/MODE now shows maximal evenness.
- The command SET ATTRIBUTE has a few new options.
- The command CALCULATE can show note names in all systems.
- Synthesizer support has been added for Yamaha XG and Roland GS & JV/XP
types (testers wanted).
- Scale files, command files and keyboard mappings can be given as
command line parameters. This means associations can be made in Windows
for the file types .scl, .cmd and .kbm.
- A new command file playwin.cmd has been added to use the Windows midi
file
player.
