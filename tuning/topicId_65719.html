I checked on Vocaloid again, and it now has a demo available. I tried
that, and it does work. Unfortunately, like a lot of demos it disables
so many features it's hard to tell if it would really be useful to me,
but it seems likely.

Here's the demo page:

http://www.zero-g.co.uk/index.cfm?articleid=802

This gives you the demo version with the usual nearly useless and
highly irritating piano roll editor. According to the documetation,
pitch bending is possible, in the usual +-8192 format, but I don't see
how to do it in the demo version. The full version is said to import
midi files, and this is a feature it is crucial to check, so of course
in their wisdom they haven't enabled it so one can see if it works or
not. As is so often the case, they don't quote a price either.

However, the demo will allow you to create 12et pieces and export them
as wav files.
