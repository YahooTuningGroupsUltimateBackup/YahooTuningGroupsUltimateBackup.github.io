We're up to 93% of our 100 MB limit on files. Yikes.

I ask that everyone please delete any old files that they have
uploaded that are no longer relevant. For instance, musical examples
that you posted years ago as a one-off thing can probably go. Anything
that you own that's 1MB + that isn't currently relevant should
probably go as well.

I also ask that people be considerate when uploading files - we don't
have much space. So if you're uploading an MP3, and that MP3 is over a
megabyte large, it's probably a good idea to host it somewhere else.

I think it might also be time to set up a site like this for file hosting:

https://www.nearlyfreespeech.net/

1 cent per megabyte-month isn't bad at all. 1 cent per GB on top of
it. If anyone knows any decent PHP software to let people have their
own accounts and upload files, I'll set something like that up.
