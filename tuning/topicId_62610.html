Scala version 2.21v is now available for OS X 10.4 Tiger. Many thanks
to Nick Dan for this who did a lot of work in getting it to compile
and making an installer. Also to Marc Sabat for testing the installer
until the problems were all fixed.
Unfortunately it will not run on 10.3 for technical reasons concerning
the X11.app from Apple.
This Mac version is now also out of the beta stage, which means
everything is supposed to work normally.
There is currently one small known problem: if you try to select a
notation system in the Select Notation System dialog with a keyboard
key, it will crash, so only use the mouse there.
Follow the installation directions on the Download page carefully,
because it's slightly different from the previous version.
http://www.xs4all.nl/~huygensf/scala/downloads.html
