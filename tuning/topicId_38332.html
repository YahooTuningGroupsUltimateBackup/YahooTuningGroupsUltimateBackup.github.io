Attachments

I wanted to listen to a chord progression, so I first tried the first
attachment (a Scala seq file); telling Scala to turn it into a midi file.
It choked on it, so I tried again with a 612-et version, but Scala
decided I'd overflowed my allotted amount of scale. So I tried the next
version, which replaced the actual JI rations with simpler ones. After
all this, the result was the attached midi file. I imagine if I'd pushed
much harder Scala would have started to complain that it ran out of midi
channels, which it sometimes will.

The question is, is this JI? I used rational numbers, but the "wrong"
ones. Would the 612-version have been JI, or only the first one? Does it
matter if Scala can't tune well enough to tell the difference?
