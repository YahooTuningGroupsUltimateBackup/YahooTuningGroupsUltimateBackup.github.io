Have there been any polls using seventh chords all the way
through? (I guess it will have to be a 5 part setting. Sorry but
I can't try this myself.) (Also, this is not advocacy of
anything, but asking genuine questions.)

In a I7-IV7-V7-I7 using all just 4:5:6:7 chords I expect nothing
special, most of all nothing dissonant. The sevenths should be
nothing but color. Not sure if this dense texture all the way
through makes musical sense without a strong melodic excuse, but
this is an
experiment, o.k.?

Now exchange the 4:7 in the V7 for a 16:9. Which will make the
better closure? The 16:9 has a dissonance that resolves into the
next chord. The 7:4 has the better (well, narrower) voice
leading.

Should a functional difference between the different sevenths
turn up, would it be possible to convert the cadence
unequivocally into a V7-I7-II#7-V7 sequence just by inserting
the right kind of 7?
