Just intonation is working in Lilypond now, with Sagittal
accidentals. For an example copied from the Sagittal
exposition, see here:

http://x31eq.com/lilypond/ratios.pdf

For the code to make it, see here:

http://x31eq.com/lilypond/lilypond-sagittal.zip

Lilypond understands ratios, so the music part starts like
this:

\tonic c
\tune #1/1 c2 \tune #3/2 g \tune #4/3 f \bar "||"
\tune #5/3 a \tune #5/4 e \tune #6/5 e \tune #8/5 a
\bar "||"
