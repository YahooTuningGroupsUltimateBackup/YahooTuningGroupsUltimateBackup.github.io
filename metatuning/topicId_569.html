Erik,

The metatuning list was set up for one purpose, and one purpose only:
to handle postings deemed 'off-topic' for the main tuning list, so
that the people and community that developed on that list would have
a place to continue a conversation/debate/rant without bothering the
list itself.

Recently, the level of dialogue (if it can be called that) has sunk
to new lows, but that isn't always the case. In any event, I'd only
suggest joining this group if there is a particular reason to follow
a thread. Otherwise, it simply functions as the place one refers to
when using the phrase "hey, would you two please take it outside?"...

Cheers,
