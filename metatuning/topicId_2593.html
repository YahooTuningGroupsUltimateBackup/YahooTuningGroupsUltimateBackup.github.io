MetaMix 1.0b3 is now available at:

http://music.columbia.edu/~jason/MetaMix/

MetaMix is a musical composition / interactive environment / computer
program / conceptual art thing. For more info, look at the web page and
the docs it links to. Or better yet, just try it out :)

This is a beta version, so there are still probably a few bugs and
a few missing features. But that's where you come in! Download it, install
it, try it out, see if it works.

MetaMix runs on Mac OS X, and, new with this release, on Windows
95/98/ME/NT/2000/XP!!! Please carefully follow the installation
instructions. (MetaMix does not run on Linux.)

Then, please e-mail me and let me know:
* what OS you use (i.e. Windows 98, Mac OS X 10.1.2, etc.)
* what happened (worked perfectly, crashed when you did X, gave error
message when you did X, would not launch, etc.)

I appreciate in advance all of your help.
